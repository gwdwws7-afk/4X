# C3 意识形态输出系统 (Ideological Output System)

> **状态**: Draft
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P2（软实力核心机制）

## Overview 概述

意识形态输出系统是"抵抗叙事"的机制化表达。瓦希德最大的武器不是导弹，而是它的故事。本系统定义意识形态行动如何消耗灰烬志，如何影响敌国民众，以及如何与军事/外交行动联动。

## Player Fantasy 玩家幻想

玩家感受到"叙事本身就是武器"。

你在贝鲁特投放了一条宣传——抵抗叙事感染率上升。当地民众开始对瓦希德产生同情，这让你在未来的外交谈判中多了一张牌。但宣传不是免费的，灰烬志消耗了，而且如果被黄金领反击为"独裁宣传"，效果会适得其反。

意识形态的胜利不是一蹴而就的——它需要持续投入，需要配合军事胜利，需要等待时机。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 意识形态行动类型

| 行动类型 | 英文 | 行动点 | 消耗 | 效果 |
|---------|------|--------|------|------|
| **官方宣传** | Official Propaganda | 1 | 灰烬志-5 | 提升目标区域抵抗叙事感染率 |
| **秘密政治支持** | Covert Political Support | 2 | 灰烬志-8，金叶-20 | 目标国内部出现亲瓦希德势力 |
| **文明交流输出** | Cultural Exchange | 1 | 灰烬志-5，社稷值-3 | 吸引温和派和世俗派 |
| **抵抗叙事峰会** | Resistance Summit | 2 | 灰烬志-10，朝贡序-5 | 提升国际道义地位 |
| **革命输出** | Revolutionary Export | 3 | 灰烬志-15 | 目标国出现武装抵抗萌芽（需军事配合） |

#### 抵抗叙事感染率

```
感染率 = 瓦希德叙事吸引力 - 黄金领反宣传 + 军事事件修正

感染率范围：0% ~ 100%

感染效果：
- < 30%：民众不支持瓦希德
- 30-60%：民众同情瓦希德
- 60-80%：民众倾向瓦希德
- > 80%：民众支持瓦希德，可触发政治变化
```

#### 意识形态与胜利路径

```
能源解放路径 - 软性配合：
- 抵抗叙事提升 → 国际道义地位上升
- 国际道义上升 → 朝贡序提升
- 朝贡序达标 → 外交解决更容易

抵抗轴心胜利路径 - 硬性支撑：
- 意识形态输出 → 目标国民众支持
- 民众支持 → 军事行动成功率提升
- 军事成功 → 节点控制巩固
```

### 意识形态行动详解

**官方宣传**：
```
效果：目标区域抵抗叙事感染率 +5%~+15%
限制：
- 每次最多影响2个区域
- 需节点为非敌对控制
成功率：基础70% + 灰烬志加成
```

**秘密政治支持**：
```
效果：目标国出现"亲瓦希德派系"
限制：
- 需要目标国关系 < 0
- 持续时间：5回合
- 被发现风险：20%（被发现则关系-10）
成功效果：目标国内部出现派系，支持率+10%
```

**文明交流输出**：
```
效果：吸引温和派和世俗派
限制：
- 目标国关系需 > -40
- 每次消耗社稷值
成功效果：目标国民众对瓦希德印象改善
```

**抵抗叙事峰会**：
```
效果：所有抵抗轴心成员国外交关系+5
限制：
- 需要至少2个灰烬众成员参加
- 需朝贡序-5
成功效果：国际道义地位上升
```

**革命输出**：
```
效果：目标国出现"武装抵抗萌芽"
限制：
- 需要军事行动配合（D1已控制目标区域）
- 需要感染率 > 60%
- 消耗大量灰烬志
成功效果：目标国稳定度下降，抵抗轴心影响力上升
```

### 意识形态与外交联动

```
意识形态输出 → 影响C1外交关系：
- 感染率 > 60%：目标国对瓦希德关系+5
- 官方宣传成功：目标国对瓦希德关系+3
- 秘密政治支持成功：目标国对瓦希德关系+10

意识形态输出 → 影响E内部政治：
- 文明交流输出：社稷值-3（内部资源分流）
- 过度宣传：可能引发"宣传疲劳"事件
```

## Formulas 公式

### 核心计算

#### 公式1：感染率变化
```
感染率变化 = 基础变化 × 灰烬志系数 × 军事修正 × 反宣传修正

基础变化：
- 官方宣传成功：+5%~+15%
- 秘密政治支持成功：+10%
- 文明交流输出：+5%
- 无行动：+0%（自然衰减）

灰烬志系数：
= (灰烬志 / 100) × 1.0
灰烬志 > 50：系数 > 0.5（效果更好）
灰烬志 < 30：系数 < 0.3（效果较差）

军事修正：
- 近期军事胜利（<3回合）：+10%
- 近期军事失败（<3回合）：-10%
- 目标区域有瓦希德控制节点：+5%

反宣传修正：
- 黄金领反宣传：-5%~-15%
- 敌方意识形态中心活跃：-10%
```

#### 公式2：意识形态行动成功率
```
成功率 = 基础成功率 + 灰烬志加成 - 敌方抵抗

基础成功率：
- 官方宣传：70%
- 秘密政治支持：60%
- 文明交流输出：65%
- 抵抗叙事峰会：75%
- 革命输出：50%

灰烬志加成：
= (灰烬志 - 30) / 200
灰烬志 = 30：+0%
灰烬志 = 80：+25%
灰烬志 = 50：+10%

敌方抵抗：
- 敌方有反宣传能力：-10%
- 目标国关系敌对：-15%
- 目标国稳定度高：-10%
```
