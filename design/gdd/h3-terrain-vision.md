# H3 地形与视野系统 (Terrain and Vision System)

> **状态**: Draft
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P2（战场深度系统）

## Overview 概述

地形与视野系统定义战场地形对军事行动的影响，以及视野范围对信息获取的作用。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 地形类型

| 地形 | 英文 | 防御加成 | 移动成本 |
|------|------|---------|---------|
| **平原** | Plains | 0% | 1 |
| **山地** | Mountains | +20% | 2 |
| **沙漠** | Desert | -10% | 1.5 |
| **城市** | Urban | +10% | 1 |
| **水域** | Water | N/A | N/A |

#### 地形对军事行动的影响

```
地形影响公式：
防御效率 = 基础效率 × (1 + 地形防御加成)

示例：
- 瓦希德高原（山地）：不对称防御效率 = 80% × 1.2 = 96%
- 两河联邦（平原）：不对称防御效率 = 80% × 1.0 = 80%
- 贝鲁特（城市）：不对称防御效率 = 80% × 1.1 = 88%
```

#### 视野系统

```
视野范围 = 基础视野 × 地形修正 × 科技修正

基础视野：3节点半径

地形修正：
- 平原：×1.0
- 山地：×1.3（视野更远）
- 沙漠：×0.9
- 城市：×0.8（建筑遮挡）

科技修正：
- 基础雷达：×1.0
- 先进雷达：×1.2
- 卫星侦察：×1.5
```

#### 迷雾与视野

```
F1情报系统迷雾等级：
- 3级（全雾）：视野外区域
- 2级：视野边缘区域
- 1级：视野内但未侦察
- 0级：已侦察

视野内但未侦察（1级）：
- 可以看到节点类型
- 看不到归属
- 看不到详细数值
```
