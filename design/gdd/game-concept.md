# 卡牌契约 (Contract Cards) - 游戏概念文档

> **状态**: 概念确定
> **版本**: 1.0
> **日期**: 2026-03-25
> **作者**: GDC Brainstorm

---

## 概述

**「卡牌契约」** 是一款买断制的2D卡牌构筑游戏。玩家通过开牌库、繁殖契约兽、合成卡牌来构建自己的套牌，在炉石like的单人對战中验证自己的收集与策略。

**一句话描述**: "收集、繁殖、合成你的完美契约兽，在单人对战中证明你的套牌。"

---

## 核心幻想 (Core Fantasy)

玩家是一个契约师，拥有召唤和培养契约兽的能力。通过探索牌库、繁殖稀有契约兽、合成强力卡牌，打造独特且强大的套牌，在赛季对战中登上顶峰。

**玩家感受**: 收集的成就感 + 策略的满足感 + 养育的投入感

---

## 核心循环

### 30秒循环 (瞬间决策)

```
抽牌 → 思考手牌 → 出牌/保留 → 场面变化
```

- 每回合抽1张牌
- 决定是否使用契约兽/技能牌
- 场面随出牌变化（我方契约兽/对方场面）
- 对战节奏快，单局5-15分钟

### 5分钟循环 (单局策略)

```
开局起手 → 资源管理 → 场面控制 → 斩杀窗口
```

- 开局7张手牌，选择保留/换牌
- 水晶/能量管理（每回合递增）
- 场面控制：铺场 vs 解场 vs 打脸
- 寻找斩杀机会

### 长期循环 (收集养成)

```
获取卡牌 → 培养/繁殖 → 合成更强卡 → 完善套牌 → 赛季对战
```

**获取途径**:
- 牌库开包（游戏内货币，不付费）
- 繁殖获得新契约兽
- 合成获得强力卡牌
- 任务奖励

**赛季目标**:
- 通过对战获得赛季点数
- 提升段位（青铜→白银→黄金→钻石→大师）
- 赛季重置，提供重复目标

## MDA框架分析

### 美学 (Aesthetics)

| 美学 | 优先级 | 实现方式 |
|------|--------|----------|
| **挑战 (Challenge)** | P0 | 炉石like对战深度，克制系统 |
| **发现 (Discovery)** | P1 | 卡牌收集，繁殖遗传的神秘感 |
| **表达 (Expression)** | P1 | 套牌构筑的个性，繁殖路线选择 |
| **叙事 (Fantasy)** | P2 | 契约兽的背景故事，赛季叙事 |

### 动力学 (Dynamics)

- **资源管理**: 水晶、卡牌、手牌
- **场面控制**: 铺场、解场、交换
- **克制系统**: 属性/类型相互克制
- **繁殖遗传**: 外观+属性的策略性传承

### 机制 (Mechanics)

- **卡牌类型**: 契约兽(随从)、技能牌(法术)、场地牌
- **对战机制**: 回合制，攻击/防御，召唤费用
- **繁殖系统**: 两张卡牌繁殖，产出新卡（遗传属性）
- **合成系统**: 消耗卡牌合成更高级卡牌
- **赛季系统**: 段位匹配，奖励重置

## 玩家动机 (Self-Determination Theory)

| 需求 | 实现 | 玩家感受 |
|------|------|----------|
| **自主 (Autonomy)** | 多样化套牌构筑路线，繁殖选择自由 | "我的套牌我做主" |
| **胜任 (Competence)** | 策略深度，对战获胜反馈，段位提升 | "我的策略起作用了" |
| **投入 (Relatedness)** | 契约兽养成投入感，收集成就感 | "这是我独一无二的契约兽" |

## 设计支柱 (Game Pillars)

### P0: 收集驱动

**定义**: 多种途径获取卡牌（繁殖/合成/牌库），每种途径都有独特价值

**设计测试**: "如果这个功能不增加获取新卡的途径或增加卡牌价值，它属于核心吗？"

**实现**:
- 牌库：稳定获取基础卡
- 繁殖：获取独特遗传卡
- 合成：获取强力限定卡

### P1: 繁殖遗传策略

**定义**: 繁殖不只是随机，是有策略的遗传系统——外观+属性可预测地遗传

**设计测试**: "这个繁殖规则让玩家觉得'我设计了这个组合'吗？"

**实现**:
- 父母属性影响子代数值范围
- 父母外观有一定概率遗传
- 稀有遗传：隐藏属性/外观突变

### P2: 对战术可弥补卡组差距

**定义**: 策略可以弥补卡组差距，新手即使卡不全也能通过策略赢

**设计测试**: "新手玩家的策略能否在对抗不完整卡组时获得胜利？"

**实现**:
- 卡牌强度有上限
- 克制系统让弱势卡有机会
- 匹配系统考虑卡组完整度

---

## 反支柱 (Anti-Pillars)

| 反支柱 | 原因 |
|--------|------|
| **不付费抽卡** | 买断制，所有卡牌通过游戏内途径获取 |
| **不数值碾压** | 策略可以弥补卡组差距 |
| **不对战变成数学题** | 对战应该有趣，不是计算考试 |
| **不强制重复内容** | 每个赛季有新鲜感，不是重复劳动 |

## 玩家类型 (Bartle Taxonomy)

| 类型 | 优先级 | 吸引点 |
|------|--------|--------|
| **收集者 (Collectors)** | P0 | 图鉴完成，稀有繁殖 |
| **建筑师 (Builders)** | P0 | 套牌构筑，繁殖路线 |
| **探索者 (Explorers)** | P1 | 新卡效果，组合发现 |
| **杀手 (Killers)** | P1 | 对战胜利，段位冲刺 |

**主要受众**: 收集者 + 建筑师（套牌构筑是核心乐趣）

**次要受众**: 探索者（发现新组合）+ 杀手（段位追求）

## 卡牌设计

### 卡牌类型

| 类型 | 数量 | 说明 |
|------|------|------|
| **契约兽** | ~60种 | 核心卡牌，有属性/类型/技能 |
| **技能牌** | ~40张 | 法术，有即时效果 |
| **场地牌** | ~20张 | 战场环境效果 |

### 属性系统

| 属性 | 克制 | 被克制 |
|------|------|--------|
| 火 | 草 | 水 |
| 水 | 火 | 雷 |
| 雷 | 水 | 草 |
| 草 | 雷 | 火 |
| 暗 | 光 | 无 |
| 光 | 暗 | 无 |

### 契约兽类型定位 (街霸式性能差异)

| 定位 | 特点 | 适合策略 |
|------|------|----------|
| **攻击型** | 高攻击，低生命 | 速攻 |
| **防御型** | 低攻击，高生命 | 拖延 |
| **平衡型** | 平均属性 | 中速 |
| **控制型** | 特效强，费用高 | 控制 |
| **支援型** | 为其他卡提供增益 | 组合 |

### 卡牌稀有度

| 稀有度 | 获取方式 | 概率 |
|--------|----------|------|
| 普通 | 牌库/合成 | 常见 |
| 优秀 | 牌库/繁殖 | 一般 |
| 稀有 | 繁殖 | 少见 |
| 史诗 | 繁殖/合成 | 稀有 |
| 传说 | 合成保底 | 极稀有 |

## 繁殖系统

### 基础规则

- 两张同类契约兽可繁殖
- 子代继承父母类型定位
- 子代属性在父母属性范围内浮动
- 子代有概率遗传父母外观特征

### 遗传机制

| 遗传项 | 机制 |
|--------|------|
| **基础属性** | 父母数值范围的平均值±随机 |
| **稀有属性** | 10%概率出现父母没有的隐藏属性 |
| **外观** | 60%概率像父，40%概率像母（含变异） |
| **技能** | 基础技能必定遗传，特殊技能有概率 |

### 繁殖限制

- 同类契约兽才能繁殖
- 有繁殖冷却时间
- 部分传说卡不可繁殖

---

## 合成系统

### 合成规则

- 消耗3张同类型卡牌
- 必定获得下一稀有度卡牌
- 有保底机制（5次不出传说必出）

### 合成表

| 投入 | 产出 |
|------|------|
| 3×普通 | 1×优秀 |
| 3×优秀 | 1×稀有 |
| 3×稀有 | 1×史诗 |
| 3×史诗 | 1×传说 |

## 赛季系统

### 赛季结构

| 赛季 | 时长 | 内容 |
|------|------|------|
| **赛季周期** | 2个月 | 一个完整赛季 |
| **重置** | 新赛季 | 段位重置，卡牌保留 |

### 段位

| 段位 | 所需点数 | 奖励 |
|------|----------|------|
| 青铜 | 0 | 基础卡牌包 |
| 白银 | 1000 | 优秀卡牌包 |
| 黄金 | 3000 | 稀有卡牌包 |
| 钻石 | 6000 | 史诗卡牌包 |
| 大师 | 10000 | 传说卡牌包 |

### 赛季奖励

- 段位奖励：卡牌包
- 收集成就：完成图鉴解锁奖励
- 首次到达段位：特殊卡背/头像

## 技术方案

### 引擎

- **Unity 2023.2 LTS**
- 2D项目，轻量级
- UI: Unity UI (UGUI)

### 美术方案

- **OpenAI DALL-E/GPT-4o生成卡牌原画**
- UI素材：AI生成 + 程序化
- 无需动画：卡牌对战只需静态立绘 + 简单特效

### 卡牌数据

- JSON配置文件
- 稀有度/属性/技能全部数据驱动
- 便于后续扩展

### 存档系统

- 本地存档（Steam云同步可选）
- 卡牌收集进度
- 赛季段位
- 套牌保存

## 风险评估

| 风险 | 严重度 | 缓解方案 |
|------|--------|----------|
| OpenAI美术风格不一致 | 中 | 统一Prompt模板，人工筛选 |
| 卡牌平衡困难 | 高 | 简化属性系统，强力测试 |
| 内容消耗过快 | 中 | 繁殖/合成提供深度 |
| 开发延期 | 高 | 严格MVP优先，砍功能 |
