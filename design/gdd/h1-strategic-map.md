# H1 战略地图系统 (Strategic Map System)

> **状态**: Designed
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P0/P1/P2/P3（所有支柱的视觉载体）

## Overview 概述

战略地图是游戏的核心视觉界面，以卫星地形为底、政治新闻标注为表的方式呈现整个游戏世界。它不是传统4X的六角格或网格，而是一张以战略节点为核心的地缘政治局势图。

地图展示了势力范围、咽喉要道、能源节点和贸易枢纽的位置关系。玩家通过地图了解全局态势，通过点击节点查看详情，通过战争迷雾感知信息不对称带来的压力。

## Player Fantasy 玩家幻想

玩家感受到"身临地缘政治局势中心"的现场感。

地图不是一张静态的战局图，而是一张"实时演化的政治纪录片截图"。半透明的势力范围在地图上缓慢移动，红色警示标注着危机热点，淡金色的航道线标注着能源运输命脉。当玩家在情报不足的区域看到一团灰色迷雾时，那种"我不知道那里有什么，但我知道那里一定有危险"的不确定性，就是地图给玩家的核心感受。

**设计目标**：玩家第一眼看到地图就知道"我在这里，敌人在那里，机会在那里"——而不是需要阅读大量文字才能理解局势。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 地图表现形式：区域节点地图

地图由以下元素构成：

| 元素 | 说明 |
|------|------|
| **战略节点** | 地图上的关键位置，代表能源产地、咽喉要道、贸易枢纽等 |
| **势力范围** | 半透明色块叠加，表示各势力在区域的影响力 |
| **节点连线** | 节点之间的关系线，表示贸易路线、军事部署、影响力投射 |
| **地形底图** | 卫星风格地形图像，显示山脉、沙漠、海洋、河流 |
| **迷雾层** | 遮挡未知区域的雾层 |

#### 节点类型

| 节点类型 | 图标 | 颜色 | 代表地点 |
|----------|------|------|---------|
| **能源节点** | 火焰图标 | 暗红色 | 瓦希德高原、波斯湾、北境能源产地 |
| **咽喉要道** | 闪电图标 | 金色 | 火海咽喉（霍尔木兹海峡）、红海要道 |
| **贸易枢纽** | 天平图标 | 绿色 | 两河联邦贸易中心、金帐合众港口 |
| **意识形态中心** | 星月图标 | 暗红金色 | 德黑兰、大马士革、贝鲁特 |

#### 势力范围显示

每个势力在地图上有一个势力范围色块：
- 瓦希德：暗红 + 金色
- 黄金领：褪色金 + 帝国蓝
- 圣火序：银白 + 六芒金
- 金帐合众：沙漠金 + 部落绿
- 中立/未知区域：灰色

势力范围不是精确边界，而是"影响力投影"的模糊区域。

#### 战争迷雾

| 区域状态 | 显示效果 |
|----------|---------|
| **己方/盟国** | 完全显示，势力范围和节点状态清晰可见 |
| **敌方控制但已知** | 半透明显示，显示但不显示详细信息 |
| **敌方控制且未知** | 战争迷雾遮挡，显示为灰色/模糊区域 |
| **中立区域** | 部分显示，需要侦察才能看到节点详情 |

#### 地图操作

| 操作 | 行为 |
|------|------|
| **缩放** | 鼠标滚轮缩放，范围50%-200% |
| **平移** | 鼠标右键拖拽 |
| **节点悬停** | 显示简要信息浮窗（节点名称、归属、价值） |
| **节点点击** | 打开节点详情面板（由K UI框架负责） |
| **右键点击** | 打开快速操作菜单（军事部署/情报侦察/外交互动） |

#### 地图区域划分

地图显示的核心区域（对应游戏背景）：

| 区域 | 范围 | 主要节点 |
|------|------|---------|
| **火海** | 波斯湾 | 火海咽喉（咽喉要道）、瓦希德高原（能源节点） |
| **黎凡特** | 叙利亚/黎巴嫩 | 大马士革（意识形态中心）、贝鲁特（贸易枢纽） |
| **红海** | 也门/吉布提 | 燎原团控制区、航道咽喉（咽喉要道） |
| **两河** | 伊拉克 | 两河联邦（贸易枢纽）、能源节点 |
| **北境** | 中亚/高加索 | 北境共主影响区、侧翼通道 |
| **东土** | 中亚东部 | 东土商盟影响区、贸易节点 |

### States and Transitions 状态与转换

地图系统本身不维护独立状态机，它维护的是地图的视觉渲染状态。

地图渲染状态变化由以下事件触发：
1. 玩家缩放/平移 → 更新视口状态
2. 回合推进 → 势力范围更新、迷雾刷新
3. 玩家执行情报行动 → 迷雾移除/添加
4. 节点归属变化 → 势力范围色块更新

### Interactions with Other Systems 与其他系统的交互

#### 上游依赖

本系统无上游依赖（Foundation层）。

#### 下游依赖

| 系统 | 依赖性质 | 依赖内容 |
|------|---------|---------|
| H2 战略节点系统 | 硬依赖 | 节点的位置、类型、归属数据 |
| K UI框架 | 硬依赖 | 节点详情面板、快速操作菜单、势力标注UI |
| C1 外交关系 | 软依赖 | 势力范围颜色和透明度 |
| D1 军事行动 | 软依赖 | 军事部署可视化、部队图标位置 |
| E 内部政治 | 软依赖 | 内部稳定度变化影响势力范围亮度 |
| F1 情报系统 | 软依赖 | 迷雾层级的更新逻辑 |

#### 地图与UI框架的分工

| 职责 | 归属 |
|------|------|
| 地图渲染（地形底图、势力范围、节点图标、连线） | H1 战略地图 |
| 节点点击检测和视口管理 | H1 战略地图 |
| 节点详情面板（数据显示） | K UI框架 |
| 快速操作菜单（军事/情报/外交） | K UI框架 |
| 势力标注文字和图标 | K UI框架 |
| 迷雾层的渲染 | H1 战略地图 |
| 迷雾数据的逻辑（哪里有雾） | F1 情报系统（通过接口） |

#### 数据接口

本系统向其他系统提供：
- `getVisibleNodes(faction)` — 获取某势力可见的节点列表
- `getFogState(node)` — 获取某节点的迷雾状态（无雾/半雾/全雾）
- `onNodeClicked(node)` — 节点被点击事件（通知K UI打开详情）
- `onNodeHovered(node)` — 节点被悬停事件（通知K UI显示简要信息）

本系统从其他系统接收：
- `setFogState(node, state)` — 由F1情报系统调用，更新迷雾状态
- `setNodeOwnership(node, faction)` — 由下游系统调用，更新节点归属

## Formulas 公式

### 核心计算

#### 公式1：势力范围透明度
```
势力范围可见度 = 基础透明度 × 稳定度系数
基础透明度：40%（不透明势力范围）
稳定度系数 = 社稷值 / 100
当社稷值=70时：可见度 = 40% × 0.7 = 28%
```

#### 公式2：迷雾遮挡等级
```
迷雾等级 = 3 - 已侦察等级
3 = 完全迷雾（灰色遮挡）
2 = 高级迷雾（只能看到节点类型，看不到归属）
1 = 低级迷雾（能看到归属，看不到详细数值）
0 = 无雾（完全可见）
```

#### 公式3：地图缩放范围
```
最小缩放 = 50%
最大缩放 = 200%
缩放步进 = 10%
```

## Edge Cases 边界情况

| 边界情况 | 处理方式 |
|---------|---------|
| 节点归属变化时地图未更新 | 每回合结算时强制刷新势力范围 |
| 迷雾数据损坏 | 默认为"高级迷雾"（2级） |
| 节点数据缺失 | 显示为"未知节点"，灰色图标 |
| 大量节点同时变化 | 合并为一次渲染更新，避免卡顿 |
| 玩家快速缩放/平移 | 限制更新频率（最高30fps渲染） |

## Tuning Knobs 调参旋钮

| 旋钮 | 默认值 | 可调范围 | 调整后果 |
|------|--------|---------|---------|
| 势力范围基础透明度 | 40% | 20%-60% | 过高→地图太花；过低→看不清势力范围 |
| 最小缩放 | 50% | 30%-70% | 过高→无法总览全局；过低→看不清节点 |
| 最大缩放 | 200% | 150%-300% | 过高→失去概览感；过低→无法看细节 |
| 迷雾刷新频率 | 每回合 | 每回合/手动 | 自动刷新用户体验更好 |
| 节点图标大小 | 32px | 24-48px | 影响可读性和视觉密度 |
| 势力范围边缘模糊度 | 20px | 10-40px | 影响势力交界处的可读性 |

**默认值已确定。**

## Visual/Audio Requirements 视觉要求

### 视觉风格

**卫星地形底图**：
- 沙漠色调为主（中东/中亚背景）
- 山脉用深色阴影标注
- 河流和海岸线用细白线标注
- 云层用淡淡的白色叠加（不遮挡地形）

**势力范围色块**：
- 每个势力有独特的主色调（参考势力设计文档）
- 半透明叠加，边缘模糊
- 随稳定度（社稷值）变化透明度

**迷雾层**：
- 纯灰色渐变遮挡
- 边缘有柔和羽化效果

### 节点图标

- 简洁几何图标（火焰/闪电/天平/星月）
- 24-32px大小
- 有势力颜色边框
- 悬停时放大10%
