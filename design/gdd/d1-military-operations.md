# D1 军事行动系统 (Military Operations System)

> **状态**: Designed
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P0（不对称博弈核心玩法）, P1（代价系统驱动）

## Overview 概述

军事行动系统是玩家使用军事力量实现战略目标的系统。瓦希德的军事战略核心是"不对称战争"——正面战争必输，但侧翼可以赢。本系统定义军事行动的类型、消耗、效果、以及与节点争夺的联动。

玩家的军事目标不是"占领敌人首都"，而是控制关键节点、切断敌方补给线、维持代理人网络、通过咽喉威慑换取政治筹码。每一次军事行动都有代价——战械消耗、社稷值影响、封锁升级风险。玩家需要在有限的军事资源下做出最优决策。

## Player Fantasy 玩家幻想

玩家感受到"以小博大"的智慧和"四两拨千斤"的刺激。

用1个行动点支持的代理人行动，在贝鲁特制造一次"意外冲突"，消耗2点战械，但可能让瓦希德控制贝鲁特——而贝鲁特的贸易枢纽金叶收入是2/回合。不对称防御消耗3点战械，但能守住瓦希德高原，保住8火油/回合的产出。如果咽喉被威胁（B2封锁升级），玩家可以用特种部队进行一次"打了就跑"的精确打击，切断敌方一条补给线，自己几乎不受损失。

军事行动不是砸钱堆兵，而是精准计算投入产出比。每次行动前玩家都会想：我投入多少？我能得到什么？代价是什么？风险有多大？

## Detailed Design 详细设计

### Core Rules 核心规则

#### 军事行动类型

| 行动类型 | 英文 | 行动点 | 战械消耗 | 风险等级 | 适用场景 |
|---------|------|--------|---------|---------|---------|
| **代理人行动** | Proxy Action | 1-2 | 2-4 | 低 | 支持燎原团/灰烬众，以小博大 |
| **特种部队** | Special Forces | 2 | 3 | 中 | 精确打击、情报行动、破坏基础设施 |
| **咽喉威胁** | Chokepoint Threat | 2 | 1 | 中 | 威胁关闭咽喉，勒索全球能源市场 |
| **不对称防御** | Asymmetric Defense | 3 | 4 | 中 | 利用地形和纵深消耗对手 |
| **核威慑展示** | Nuclear Deterrence Display | 4 | 0 | 高 | 阻止全面战争，威慑敌方升级 |
| **全面战争** | Total War | 5+ | 8+ | 极高 | 几乎不可能获胜的选项 |

#### 军事行动消耗详解

**代理人行动**：
```
行动点：1-2（取决于规模）
战械消耗：2-4（取决于目标）
成功效果：目标节点争夺点数+20~40（取决于投入）
失败效果：消耗的50%战械损失，灰烬志-5
适用目标：贝鲁特、大马士革（意识形态中心/贸易枢纽）
```

**特种部队**：
```
行动点：2
战械消耗：3（打击）/ 1（情报）
可选子类型：
- 精确打击：消耗3战械，对节点造成40控制点伤害
- 情报行动：消耗1战械，获得目标节点详细情报（消除迷雾）

成功效果：打击目标争夺点-40；情报目标迷雾消除
失败效果：特种部队损失，社稷值-3
适用目标：任意节点
```

**咽喉威胁**：
```
行动点：2
战械消耗：1（维持存在）
成功效果：咽喉进入"受压"状态，路线效率-30%；B2封锁升级风险+40%
失败效果：咽喉进入敌方控制，路线完全中断
适用目标：火海咽喉、红海咽喉

特殊机制：
- 咽喉威胁是"展示存在"而非"攻击"
- 可选择"升级威胁"：增加战械投入，提升封锁压力
- 可选择"撤回威胁"：解除威胁，避免封锁升级
```

**不对称防御**：
```
行动点：3
战械消耗：4
成功效果：防守节点稳定性+10，争夺点+30
失败效果：节点争夺点-20，社稷值-5
适用目标：瓦希德高原、火海咽喉

防御加成：
- 地形加成：防御方+20%效率
- 纵深加成：若相邻节点为友方，+10%效率
```

**核威慑展示**：
```
行动点：4
战械消耗：0
成功效果：敌方全面战争选项被锁定1回合；B2封锁升级风险-20%
失败效果：朝贡序-15；触发D4核威慑系统联动
适用目标：无（全局效果）

[AI决策影响]：
核威慑展示改变敌方AI行动权重：
- 敌方"直接攻击"权重：0.4 → 0.1（降低）
- 敌方"外交施压"权重：0.3 → 0.5（提升）
- 封锁升级审议时，AI更倾向"暂缓升级"
- 军事行动选择时，AI更倾向"代理人"而非"直接攻击"

限制：
- 每10回合只能使用1次
- 使用后灰烬志-10，朝贡序-15（展示代价）
```

**全面战争**：
```
行动点：5+
战械消耗：8+（按规模）
成功效果：目标节点强制占领
失败效果：灾难性——社稷值-20，封锁全面升级，灰烬志-15
适用目标：任意节点

警告：全面战争是"自杀选项"，只有在极端情况下才考虑

[MVP简化版实现]：
- 全面战争选项存在：显示消耗和极高风险
- 触发后：直接跳转"灾难性后果"处理（社稷值-20，封锁全面升级等）
- 不做：完整的战争引擎、多方兵力计算、长期战争延续
[Alpha阶段]：细化战争引擎，支持战争延续多回合
```

### 行动点分配

每回合军事阶段有**3点基础行动点**（来自A1回合主循环），另有**2点通用行动点**可自由分配到任意阶段。

```
军事阶段行动点：3点（专用）
通用行动点：2点（可分配到军事/外交/情报等）

行动点使用示例：
- 代理人行动(2点) + 特种部队(2点) → 需要5点，超出3点军事专用上限
  → 解决方案：使用1点通用点补充
- 不对称防御(3点) + 咽喉威胁(2点) → 需要5点
  → 解决方案：使用2点通用点补充
```

### 军事行动效果链

```
玩家执行军事行动
    ↓
消耗行动点和战械
    ↓
判断行动类型
    ↓
计算直接效果
    ├── 节点控制点变化（+/-）
    ├── 节点稳定性变化（+/-）
    ├── 迷雾状态变化（如有情报行动）
    └── 路线效率变化（如咽喉威胁）
    ↓
触发下游系统联动
    ├── H2节点系统：更新控制点/归属
    ├── B3贸易网络：重新计算路线效率
    ├── E内部政治：社稷值变化
    ├── C1外交关系：关系变化
    ├── B2封锁系统：封锁升级风险评估
    └── I1事件系统：可能触发叙事事件
```

### 行动点消耗规则

```
消耗规则：
- 行动点不跨回合累积
- 未使用的行动点不会保留到下回合
- 行动一旦开始，消耗的行动点不返还（即使失败）

例外情况：
- 行动因"战械不足"无法执行：行动点返还
- 行动因"外部事件中断"：部分返还（视情况）
```

### 军事行动与节点争夺

#### 争夺点数积累

```
节点争夺点数 = Σ(各方本回合投入)

控制门槛：
- 初期争夺：1/3门槛（33点）
- 深度争夺：2/3门槛（66点）
- 占领待定：100点（等待回合结算）

回合结算（D5战争结算）：
- 若某方 > 100点且领先对手30点：获得节点控制权
- 若势均力敌（差距<30点）：维持争夺状态
```

#### 争夺状态转换

| 状态 | 控制点范围 | 产出效率 | 说明 |
|------|-----------|---------|------|
| 中立 | 0 | 100% | 无势力控制 |
| 初期争夺 | 1-66 | 50% | 军事存在，产出减半 |
| 深度争夺 | 67-99 | 0% | 产出中断，稳定性下降 |
| 势力控制 | >100 | 100% | 某势力稳定控制 |
| 争夺待定 | =100 | — | 等待回合结算判定 |

#### 节点争夺优先级

当多个节点同时处于争夺状态时：

```
争夺优先级 = 战略价值 × (1 - 当前控制比例)

示例：
- 火海咽喉：战略价值100，当前控制60%
  → 优先级 = 100 × (1-0.6) = 40
- 贝鲁特：战略价值60，当前控制30%
  → 优先级 = 60 × (1-0.3) = 42

系统自动优先处理高优先级节点的争夺
```

### 军事行动与B2封锁联动

```
军事行动触发封锁升级风险评估：

风险值 = Σ(该行动类型风险值)
行动类型风险值：
- 代理人行动：+15
- 特种部队：+20
- 咽喉威胁：+40
- 不对称防御：+10
- 核威慑展示：-20（降低风险）
- 全面战争：+100

B2封锁系统收到风险值后：
1. 将风险值加入当前封锁压力
2. 若超过阈值：进入"封锁升级审议"
3. 黄金领AI决策：升级/警告/暂缓
```

## Formulas 公式

### 核心计算

#### 公式1：行动成功率
```
行动成功率 = 基础成功率 + 加成 - 惩罚

基础成功率：
- 代理人行动：70%
- 特种部队：60%
- 咽喉威胁：50%
- 不对称防御：80%（防御方优势）
- 核威慑展示：100%（展示而非攻击）
- 全面战争：30%

加成来源：
- 战械充足（>10）：+10%
- 灰烬志>50：+10%
- 相邻友好节点：+10%
- 敌方节点稳定性<30：+15%

惩罚来源：
- 社稷值<50：-10%
- 敌方防御加成：-20%（攻击咽喉时）
- 敌方地形优势：-15%
```

#### 公式2：行动效果量
```
效果量 = 基础效果量 × 成功率系数 × 加成系数

成功率系数：
- 成功（>100%）：1.0
- 部分成功（70-99%）：0.7
- 勉强成功（50-69%）：0.4
- 失败（<50%）：0（甚至负效果）

加成系数：
- 战械超量投入：每超1点 +0.1
- 情报优势（敌方迷雾）：+0.2
- 联盟协同：+0.15
```

#### 公式3：封锁升级风险
```
封锁升级风险 = Σ(行动风险值) + Σ(情境修正) - Σ(缓释手段)

行动风险值：
- 代理人行动：+15
- 特种部队：+20
- 咽喉威胁：+40
- 不对称防御：+10
- 全面战争：+100

情境修正：
- 行动目标为能源节点：+10
- 行动目标为咽喉要道：+20
- 敌方正处于内乱：-20

缓释手段：
- 外交铺垫（提前沟通）：-30%
- 强硬反制（多线施压）：+50%
- 核威慑展示同时进行：-40%
```

#### 公式4：不对称防御效率
```
防御效率 = 基础效率 × 地形系数 × 纵深系数 × 稳定度系数

基础效率 = 80%（防御方基础成功率）

地形系数：
- 山地/高原地形：1.2
- 平原：1.0
- 城市：0.9

纵深系数：
- 相邻节点均为友方：1.15
- 相邻节点有一方中立：1.0
- 无相邻友方节点：0.85

稳定度系数 = 节点稳定性 / 100
```

## Edge Cases 边界情况

| 边界情况 | 处理方式 |
|---------|---------|
| 战械不足仍尝试行动 | 行动点返还，UI显示"战械不足" |
| 行动失败但战械已消耗 | 50%战械损失，其余不返还 |
| 同时攻击多个节点 | 每个节点独立计算，按优先级分配行动点 |
| 咽喉威胁被反制 | 咽喉控制权可能易手，路线效率变为0 |
| 全面战争失败 | 灾难性后果：社稷值-20，封锁全面升级，灰烬志-15 |
| 军事行动触发多方反应 | 按顺序结算：先节点争夺，再封锁评估，最后外交反应 |
| 行动失败但战械已消耗 | 50%战械损失，其余不返还；代理人有损失体现为资源损耗 |
| 连续军事失败 | 社稷值累计下降，触发E系统派系危机 |
| 行动点不足 | 行动不可选，UI显示所需行动点和缺口 |

## Data Interfaces 数据接口

### D1向其他系统提供的接口

```typescript
interface MilitarySystem {
    // 执行军事行动
    executeMilitaryAction(
        actionType: MilitaryActionType,
        targetNode: NodeId,
        options: ActionOptions
    ): ActionResult
    
    // 获取当前军事状态
    getMilitaryState(): MilitaryState
    
    // 获取可用行动列表（基于当前资源）
    getAvailableActions(): AvailableAction[]
    
    // 获取节点争夺状态
    getContestStatus(nodeId: NodeId): ContestStatus
    
    // 评估封锁升级风险
    assessBlockadeRisk(action: MilitaryAction): number
}

interface MilitaryState {
    availableActionPoints: number
    totalActionPoints: number
    armsResource: number
    pendingActions: PendingAction[]
    contestedNodes: ContestStatus[]
}

interface ActionResult {
    success: boolean
    partialSuccess: boolean
    nodeControlPointDelta: number
    resourceConsumed: ResourceCost
    downstreamEffects: DownstreamEffect[]
    blockadeRiskIncrease: number
    narrativeEvents: string[]
}
```
