> **状态**: LOCKED

## Overview 概述

A1 负责回合推进与阶段顺序执行，是回合/时间/AP模型的主驱动系统。

## Core Rules 核心规则

### 阶段顺序与AP

| 索引 | 阶段 | 基础AP |
|---|---|---|
| 0 | 外交 | 2 |
| 1 | 战略 | 2 |
| 2 | 作战 | 3 |
| 3 | 后勤 | 1 |
| 4 | 情报 | 1 |
| 5 | AI响应 | 0 |

- 每回合通用 AP：2
- 每回合总 AP：11（阶段9 + 通用2）

### 回合流程

`外交 -> 战略 -> 作战 -> 后勤 -> 情报 -> AI响应 -> 回合结算 -> 下一回合`

### AP约束

- 阶段专用 AP 不跨阶段。
- 通用 AP 可跨玩家阶段，不跨回合。
- 回合结束后 AP 清零并重置。

## Time Model 时间模型

- 1回合=6个月
- 起始：2028H1
- 最大回合：24
