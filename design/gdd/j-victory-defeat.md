# J 胜负判定系统 (Victory/Defeat System)

> **状态**: Designed
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P0（游戏终局验证）, P1（目标驱动）

## Overview 概述

胜负判定系统是游戏终局的裁判。它定义胜利条件、失败条件、胜负检测机制、以及终局处理流程。

本系统的核心设计目标是：让玩家在每回合都能看到自己与胜利目标的距离——不是模糊的"感觉快赢了"，而是清晰的进度条和下一步建议。

## Player Fantasy 玩家幻想

玩家在游戏过程中，感受到"胜利是有多条路可达的"。

每回合结算时，J系统告诉玩家："你在能源解放路线上进展到60%，距离胜利还需要3个条件。外交解决路线进展到40%，但本周期的联合国决议即将开始投票，建议集中外交资源。军事均势路线进展到20%，但风险极高，建议谨慎。"

当胜利条件逐渐达成时，玩家感受到"我选择的这条路终于走通了"——不是随机发生，而是玩家策略的积累结果。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 四种胜利路径

| 胜利类型 | 英文 | 触发条件 | 难度 | 胜利叙事 |
|---------|------|---------|------|---------|
| **能源解放** | Energy Liberation | 能源出口完全不依赖星币结算，封锁体系持续失效3年 | 高 | 「他们以为没有星币我们会死——他们错了」 |
| **军事均势** | Military Stalemate | 大规模军事冲突中迫使黄金领/圣火序接受停火协议（瓦希德未崩溃） | 极高 | 「全面战争代价太大，他们选择谈判桌」 |
| **抵抗轴心胜利** | Axis Victory | 抵抗轴心控制两河/叙利亚/黎巴嫩，圣火序被迫接受两国方案 | 高 | 「抵抗的旗帜插在了大马士革和贝鲁特」 |
| **外交解决** | Diplomatic Resolution | 联合国通过有利于瓦希德的决议，黎明协定全面恢复，封锁解除 | 中 | 「历史站在了我们这边」 |

#### 胜利条件详解

**1. 能源解放**

```
触发条件（需同时满足）：
条件A：能源出口完全不依赖星币结算
  - 商盟券渠道：每回合≥50单位能源通过商盟券结算
  - 北境本币渠道：每回合≥30单位能源通过北境银结算
  - 灰色市场：每回合≤10单位能源通过灰色市场（避免依赖）

条件B：封锁体系持续失效3年（36个月 = 24回合）
  - 封锁等级：连续24回合处于"单边封锁"以下
  - 或封锁体系因崩溃机制（B2）彻底瓦解

条件C：能源产出稳定
  - 瓦希德高原产出：每回合≥6火油
  - 或基尔库克在抵抗轴心控制下
```

**2. 军事均势**

```
触发条件（需同时满足）：
条件A：大规模军事冲突发生
  - D5判定：某次争夺中双方投入争夺点数 > 150

条件B：瓦希德未崩溃
  - 社稷值 > 30
  - 瓦希德高原仍在瓦希德控制下
  - 德黑兰（无具体节点，但作为判定条件）

条件C：敌方被迫接受停火
  - 黄金领/圣火序在某次D5结算后，争夺点数领先<30
  - 且封锁升级审议显示"暂缓"决策
  - 或触发"战争疲劳"事件（I1）

注：此胜利路径风险极高，若冲突中瓦希德崩溃则直接触发军事崩溃失败
```

**3. 抵抗轴心胜利**

```
触发条件（需同时满足）：
条件A：控制两河
  - 两河联邦（两河联邦）在瓦希德/灰烬众控制下
  - 或基尔库克+巴士拉同时在抵抗轴心控制下

条件B：控制叙利亚
  - 大马士革在抵抗轴心控制下

条件C：控制黎巴嫩
  - 贝鲁特在抵抗轴心控制下

条件D：圣火序接受两国方案
  - 与圣火序关系从敌对升至"冷淡"以上（+40以上）
  - 或签订"两国方案"协议（C2）
```

**4. 外交解决**

```
触发条件（需同时满足）：
条件A：联合国决议通过
  - 朝贡序 ≥ 70
  - 与北境共主/东土商盟关系 ≥ 友好(+60)
  - 联合国大会投票：支持率 > 60%

条件B：黎明协定恢复
  - 与黄金领关系从敌对升至"一般"以上（-40以上）
  - 或签订"黎明协定"协议（C2）

条件C：封锁解除
  - B2封锁等级降至"无封锁"或"单边封锁"
```

#### 失败条件

| 失败类型 | 英文 | 触发条件 | 失败叙事 |
|---------|------|---------|---------|
| **军事崩溃** | Military Collapse | 德黑兰（瓦希德核心区）被占领，政府流亡 | 「抵抗的最后堡垒陷落」 |
| **经济崩溃** | Economic Collapse | 能源收入归零且战略储备耗尽（持续3回合） | 「经济基础已经崩溃」 |
| **内部分裂** | Internal Division | 派系冲突失控（E系统触发） | 「国家在内部矛盾中瓦解」 |

#### 胜负检测机制

```
每回合结算流程中，J系统在以下时机检测：

检测时机1：回合开始时（快速检测）
- 检查是否有立即失败条件（如军事崩溃）

检测时机2：回合结算后（D5/B5之后）
- 检查各胜利路径进度
- 检查是否有失败条件

检测时机3：特定事件触发时
- 大规模军事行动结束后
- 封锁等级变化时
- 重大外交事件时
```

#### 进度计算方法

```
每回合，根据当前游戏状态，计算各胜利路径的完成百分比：

能源解放进度 = f(商盟券覆盖率, 北境渠道覆盖率, 封锁失效回合数)

军事均势进度 = f(战争发展程度, 敌方损失, 停火可能性)

抵抗轴心胜利进度 = f(节点控制率, 圣火序关系, 协议签订)

外交解决进度 = f(朝贡序, 黄金领关系, 封锁等级)
```

### 组合胜利机制

```
组合胜利：任意2种胜利条件同时满足 = 加速胜利

效果：
- 终局提前5-10回合
- 最终胜利动画/结算提前触发
- 叙事效果："双轨胜利"

示例：
- 能源解放(70%) + 外交解决(60%) = 可能提前触发终局
- 抵抗轴心胜利(90%) + 军事均势(50%) = 加速胜利
```

### 终局处理

```
胜利触发后的处理流程：

1. 触发胜利动画/结算界面
2. 播放对应胜利类型的叙事CG/文字
3. 展示胜利时刻的全球态势截图
4. 统计胜利数据：
   - 回合数
   - 各胜利路径完成度
   - 关键节点控制情况
   - 外交协议数量
   - 军事行动统计
5. 解锁成就/统计
6. 提供"再来一局"选项

失败触发后的处理流程：

1. 播放失败叙事CG/文字
2. 展示失败时刻的全球态势
3. 分析失败原因：
   - 哪个胜利路径接近但未完成
   - 关键失误点
4. 提供"快速重开"按钮（直接跳转剧本选择）
5. 保留"失败回顾"记录（玩家可选查看）
```

#### 提前预警机制

```
进度预警（接近胜利）：
- 进度 ≥ 60%：显示"胜利在望"提示
- 进度 ≥ 80%：显示"即将达成"警告
- UI显示：进度条变色（绿色加深）

失败风险预警（接近失败）：
- 军事崩溃风险：社稷值 < 35 时提示
- 经济崩溃风险：能源收入为0 且储备 < 5 时提示
- 内部分裂风险：E系统派系冲突激化时提示

预警触发时：
- UI显示预警图标和文字
- 结算报告首页显示预警
- 建议行动列表（基于当前状况）
```

#### 持久战终局（超时处理）

```
触发条件：回合24结束后仍无胜负

终局类型："消耗战平局"
叙事：「双方都筋疲力尽，但谁也没有倒下」

处理：
- 统计各胜利路径完成度
- 宣布最高完成度的路径为"接近胜利"
- 提供"历史评价"（赢了什么，输了什么）
- 玩家可选择"再来一局"或"接受现状"
```

## Formulas 公式

### 核心计算

#### 公式1：能源解放进度
```
能源解放进度 = (商盟券进度 × 0.4) + (北境渠道进度 × 0.3) + (封锁失效进度 × 0.3)

商盟券进度 = 
    (当前商盟券结算量 / 50) × 100%
    上限100%

北境渠道进度 = 
    (当前北境本币结算量 / 30) × 100%
    上限100%

封锁失效进度 = 
    (封锁等级降至单边以下的回合数 / 24) × 100%
    达到24回合则此子项=100%

胜利条件达成：三个子项均 ≥ 80%
```

#### 公式2：军事均势进度
```
军事均势进度 = (战争发展 × 0.4) + (敌方损失 × 0.3) + (停火可能性 × 0.3)

战争发展 = 
    if 存在大规模冲突(>150争夺点):
        = 50 + (冲突回合数 × 5)
    else:
        = 0

敌方损失 = 
    if 黄金领/圣火序失去关键节点:
        += 30
    if 封锁升级审议"暂缓"次数 > 3:
        += 20

停火可能性 = 
    if 黄金领国内政治压力 > 阈值:
        += 40
    if 战争持续 > 5回合:
        += 30
    if 瓦希德社稷值 > 50:
        += 20

胜利条件达成：总分 ≥ 80%
```

#### 公式3：抵抗轴心胜利进度
```
抵抗轴心胜利进度 = (两河控制 × 0.3) + (叙利亚控制 × 0.3) + (黎巴嫩控制 × 0.2) + (圣火序关系 × 0.2)

两河控制 = 
    if 两河联邦在抵抗轴心: 100%
    elif 基尔库克+巴士拉在抵抗轴心: 80%
    elif 任一在抵抗轴心: 40%
    else: 0%

叙利亚控制 = 
    if 大马士革在抵抗轴心: 100%
    else: 0%

黎巴嫩控制 = 
    if 贝鲁特在抵抗轴心: 100%
    else: 0%

圣火序关系 = 
    (关系值 + 80) / 160 × 100%
    关系值范围 -80 ~ +100

胜利条件达成：总分 ≥ 80%
```

#### 公式4：外交解决进度
```
外交解决进度 = (朝贡序进度 × 0.3) + (黄金领关系进度 × 0.4) + (封锁解除进度 × 0.3)

朝贡序进度 = 
    (当前朝贡序 / 70) × 100%
    上限100%

黄金领关系进度 = 
    (关系值 + 80) / 120 × 100%
    关系值范围 -80 ~ +40（+40=冷淡边界）

封锁解除进度 = 
    if 封锁等级 == 无封锁: 100%
    elif 封锁等级 == 单边封锁: 70%
    elif 封锁等级 == 多边封锁: 30%
    elif 封锁等级 == 全面封锁: 0%

胜利条件达成：总分 ≥ 80%
```

#### 公式5：组合胜利加速
```
if 两种胜利路径进度均 ≥ 80%:
    终局加速回合数 = 10 - (两路径平均进度 - 80) × 0.5
    终局提前触发
```

## Edge Cases 边界情况

| 边界情况 | 处理方式 |
|---------|---------|
| 两种胜利条件同时达成 | 触发"双轨胜利"特殊终局，取叙事优先级高的为主 |
| 胜利条件达成但立即触发失败 | 失败优先（军事崩溃优先于能源解放） |
| 达到24回合仍无胜负 | 触发"持久战"终局，按各路径完成度排名，提示玩家"可以选择继续或认和" |
| 玩家主动认输/投降 | 立即触发失败流程，但提供"最后努力"窗口 |
| 游戏中途放弃胜利路径 | 进度保持不变，可随时重新追求该路径 |

## Data Interfaces 数据接口

### J向其他系统提供的接口

```typescript
interface VictoryDefeatSystem {
    // 每回合检测胜负条件
    checkVictoryDefeat(): VictoryCheckResult
    
    // 获取各胜利路径进度
    getVictoryProgress(): VictoryProgressReport
    
    // 获取当前失败风险
    getDefeatRisk(): DefeatRiskReport
    
    // 触发终局处理
    triggerEndgame(type: 'victory' | 'defeat', details: EndgameDetails): EndgameResult
}

interface VictoryProgressReport {
    energyLiberation: PathProgress
    militaryStalemate: PathProgress
    axisVictory: PathProgress
    diplomaticResolution: PathProgress
    closestPath: string
    recommendedFocus: string
}

interface PathProgress {
    progress: number        // 0-100
    subComponents: {
        name: string
        progress: number
        status: 'on_track' | 'at_risk' | 'blocked'
    }[]
    estimatedTurnsToVictory: number | null
}
```

## Tuning Knobs 调参旋钮

| 旋钮 | 默认值 | 可调范围 | 调整后果 |
|------|--------|---------|---------|
| 胜利进度阈值 | 80% | 70%-90% | 影响胜利达成难度 |
| 组合胜利加速回合 | 10回合 | 5-15回合 | 影响终局提前量 |
| 军事崩溃社稷值门槛 | 30 | 20-40 | 影响军事崩溃判定 |
| 经济崩溃持续回合 | 3回合 | 2-5回合 | 影响经济崩溃判定 |
| 能源解放商盟券目标 | 50单位/回合 | 40-60 | 影响能源解放难度 |
| 能源解放北境目标 | 30单位/回合 | 20-40 | 影响能源解放难度 |
| 封锁失效要求回合 | 24回合 | 18-30回合 | 影响能源解放难度 |
| 最大游戏回合数 | 24回合 | — | 超过则触发持久战终局 |
| 胜利提前预警阈值 | 60%/80% | 50%-70%/75%-85% | 提示玩家胜利接近 |
| 失败风险预警阈值 | 35社稷值/5储备 | 30-40/3-8 | 提示玩家失败风险 |
| 快速重开 | 是 | — | 失败后立即重新开始 |

## Acceptance Criteria 验收标准

### 基本检测验证
- [ ] 每回合正确检测各胜利路径进度
- [ ] 失败条件正确触发
- [ ] 进度计算公式正确执行

### 胜利路径验证
- [ ] 能源解放：四个子项正确计算，进度正确汇总
- [ ] 军事均势：三个子项正确计算，进度正确汇总
- [ ] 抵抗轴心胜利：四个子项正确计算，进度正确汇总
- [ ] 外交解决：三个子项正确计算，进度正确汇总

### 组合胜利验证
- [ ] 两种胜利同时达成时触发加速终局
- [ ] 加速效果正确（终局提前5-10回合）
