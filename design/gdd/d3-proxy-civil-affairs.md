# D3 代理人民事系统 (Proxy Civil Affairs System)

> **状态**: Draft
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P2（代理人民事管理）

## Overview 概述

代理人民事系统管理灰烬众等代理人的民政事务，包括代理人控制区的治理、稳定度和民心。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 代理人控制区

```
代理人控制区 = 灰烬众或其他抵抗轴心成员控制的节点

每个代理人控制区有：
- 稳定度：0-100
- 民心：0-100
- 治理成本：每回合社稷值-1~-3
```

#### 代理人民事状态

| 状态 | 稳定度 | 民心 | 效果 |
|------|--------|------|------|
| 稳固 | >70 | >60 | 正常产出，抵抗叙事扩散 |
| 一般 | 40-70 | 40-60 | 产出-20%，无叙事扩散 |
| 动荡 | 20-40 | 20-40 | 产出-50%，可能叛乱 |
| 崩溃 | <20 | <20 | 产出归零，节点可能易手 |

#### 治理机制

```
治理投入 = 社稷值消耗

投入等级：
- 基础治理：-1社稷值，稳定度+2
- 强化治理：-2社稷值，稳定度+5
- 最高治理：-3社稷值，稳定度+8

稳定度衰减：
- 每回合自动-3（自然衰减）
- 若民心<40：额外-5
```
