# G 势力AI框架 (Faction AI Framework)

> **状态**: Designed
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P0（不对称博弈对手）, P1（AI响应驱动）

## Overview 概述

势力AI框架是所有非玩家势力（黄金领、圣火序、金帐合众、北境共主、东土商盟、灰烬众）的决策和行为系统。AI不是铁板一块——每个势力有自己的性格、目标、优先级，会根据局势做出不同反应。

本系统的核心设计目标是：让AI行为丰富且可预测，但不是"完美最优"。玩家应该能"读懂"AI的意图，通过策略影响AI行为，而不是面对一个黑盒。

## Player Fantasy 玩家幻想

玩家感受到"对手是活生生的、有自己诉求的势力"。

黄金领的AI不是一味压制瓦希德——当瓦希德威胁关闭火海咽喉时，黄金领会权衡利弊：是升级封锁，还是谈判？当金帐合众感觉被黄金领利用时，可能会偷偷与瓦希德接触。北境共主永远在计算：现在站队瓦希德利益大，还是站队黄金领利益大？

AI的决策不是随机的——有规律可循。玩家可以通过情报行动（F1）了解AI的意图，通过外交行动（C1）影响AI的偏好，通过军事行动（D1）改变AI的计算前提。

## Detailed Design 详细设计

### Core Rules 核心规则

#### AI势力架构

```
每个AI势力 = 人格模块 + 决策引擎 + 执行器

人格模块：
- 定义该势力的核心诉求、行为偏好、风险承受能力
- 决定在给定情境下"优先考虑什么"

决策引擎：
- 评估当前局势（读取游戏状态）
- 生成候选行动列表
- 根据人格权重计算行动得分
- 选择最优行动（带随机性）

执行器：
- 将决策转化为具体行动
- 与对应系统交互（D1军事、C1外交、B2封锁等）
```

#### 势力人格定义

| 势力 | 英文 | 核心诉求 | 风险承受 | 主要策略 | 对瓦希德态度 |
|------|------|---------|---------|---------|-------------|
| **黄金领** | Aurean Dominion | 维持中东霸权、美元石油体系 | 中等 | 封锁+代理人战争，忌惮直接军事冲突 | 敌对但克制 |
| **圣火序** | Order of Sacred Fire | 国家安全、区域影响力 | 低 | 先发制人防御、边境安全 | 敌对 |
| **金帐合众** | Golden Horde | 逊尼派领导权、能源财富 | 中低 | 经济外交、机会主义 | 冷淡-机会主义 |
| **北境共主** | Northern Hegemon | 军事技术出口、南部纵深 | 高 | 武器换能源、多方下注 | 利益导向 |
| **东土商盟** | Eastern Trade Alliance | 贸易利益、能源来源多样化 | 低 | 经济优先、避免冲突 | 实用友好 |
| **灰烬众** | Ash Confederacy | 抵抗叙事扩散、区域影响力 | 中 | 代理人网络、意识形态输出 | 盟友 |

#### 决策周期

```
AI决策在回合的第6阶段（敌对行动阶段）执行：

回合阶段：
1. 内政 → 2. 情报 → 3. 经济 → 4. 外交 → 5. 军事 → 6. 敌对行动
                                                              ↑
                                                      AI决策在此执行

AI决策顺序：
1. 黄金领（最强势力，优先决策）
2. 圣火序
3. 金帐合众
4. 北境共主
5. 东土商盟
6. 灰烬众（最后执行，因为与瓦希德协同）

每个势力决策时，读取：
- 当前游戏状态（回合数、封锁等级、各势力关系、节点归属）
- 自身人格（核心诉求、优先级）
- 历史记忆（上一回合的决策、玩家上回合的行动）
```

#### 决策类型

**黄金领决策类型**：

| 决策类型 | 触发条件 | 候选行动 | 选择逻辑 |
|---------|---------|---------|---------|
| **封锁升级审议** | 封锁压力>阈值 | 升级封锁/维持/警告/暂缓 | 权衡：升级代价 vs 瓦希德威胁程度 |
| **军事响应评估** | 瓦希德军事行动 | 代理人反击/直接威胁/外交施压/忍耐 | 取决于行动规模和目标 |
| **外交姿态调整** | 关系重大变化 | 施压/谈判/试探/无视 | 取决于当前关系和利益 |
| **资源分配** | 每回合 | 增加军事/增加封锁/维持现状 | 取决于国内政治（强硬派vs温和派） |

**金帐合众决策类型**：

| 决策类型 | 触发条件 | 候选行动 | 选择逻辑 |
|---------|---------|---------|---------|
| **能源政策** | 每回合 | 增产/减产/维持 | 油价波动+自身利益+外交因素 |
| **对瓦希德态度** | 关系变化 | 接近/疏远/观望 | 取决于黄金领压力和自身利益 |
| **二级封锁配合** | 黄金领施压 | 配合/部分配合/拒绝 | 评估被反制的风险和自身损失 |

**北境共主决策类型**：

| 决策类型 | 触发条件 | 候选行动 | 选择逻辑 |
|---------|---------|---------|---------|
| **武器销售配额** | 每回合 | 增加/维持/减少 | 取决于与瓦希德关系和自身库存 |
| **政治立场** | 局势重大变化 | 倾向瓦希德/中立/倾向黄金领 | 多方下注，计算利益最大化 |
| **联合国投票** | 决议相关 | 支持/反对/弃权 | 取决于交易和立场 |

**东土商盟决策类型**：

| 决策类型 | 触发条件 | 候选行动 | 选择逻辑 |
|---------|---------|---------|---------|
| **贸易渠道** | 每回合 | 扩大/维持/减少 | 取决于经济利益和被封锁风险 |
| **商盟券政策** | 局势变化 | 扩大/维持/收紧 | 自身经济利益优先 |
| **对黄金领压力** | 黄金领施压 | 硬顶/妥协/找折中 | 不值得为瓦希德得罪黄金领 |

### AI决策权重系统

```
行动得分 = Σ(因素权重 × 因素得分) + 随机项

示例（黄金领封锁升级审议）：

因素1：瓦希德威胁程度（权重0.4）
  - 瓦希德高原产出正常：0分
  - 瓦希德威胁关闭咽喉：+30分
  - 瓦希德军事行动频繁：+20分

因素2：升级封锁代价（权重0.3）
  - 盟友配合意愿高：0分
  - 盟友配合意愿低：+15分（升级代价大）
  - 全球油价波动大：+10分

因素3：当前国内政治（权重0.2）
  - 强硬派主导：-10分（倾向升级）
  - 温和派主导：+10分（倾向克制）

因素4：外交谈判可能性（权重0.1）
  - 瓦希德释放谈判信号：+20分（倾向暂缓）
  - 无谈判信号：0分

随机项：±5（保持不可完全预测）

总得分 > +30：升级封锁
总得分 -10~+30：警告
总得分 < -10：暂缓/维持现状
```

### AI人格差异

| 势力 | 风险承受 | 决策风格 | 对瓦希德策略 |
|------|---------|---------|-------------|
| 黄金领 | 中 | 权衡利弊 | 封锁为主，避免直接战争 |
| 圣火序 | 低 | 谨慎防御 | 先发制人，但不主动扩大冲突 |
| 金帐合众 | 中低 | 机会主义 | 观望为主，利益导向 |
| 北境共主 | 高 | 多方下注 | 武器换能源，两边卖好 |
| 东土商盟 | 低 | 经济优先 | 不值得冒险，利益第一 |
| 灰烬众 | 中 | 协同瓦希德 | 配合为主，但有自己诉求 |

### AI与玩家行动的响应

**响应延迟机制**：

```
AI对玩家行动的响应不是立即的——有决策周期：

玩家回合1行动 → AI在回合2的决策阶段响应
玩家回合2行动 → AI在回合3的决策阶段响应

这意味着：
- 玩家有1回合的"先手优势"
- AI的响应有滞后性
- 但AI可以"预判"玩家趋势（根据情报）

例外：
- 紧急情况（如咽喉被威胁关闭）：AI可立即响应（下一阶段而非下一回合）
- 情报系统（F1）可以提前揭示AI的预判
```

**玩家行动对AI的影响**：

| 玩家行动 | 对黄金领影响 | 对金帐合众影响 | 对北境共主影响 |
|---------|-------------|---------------|---------------|
| 军事行动（敌对目标） | 封锁升级风险+20 | 评估是否配合封锁 | 可能增加武器销售 |
| 军事行动（友好目标） | 关系恶化 | 担心卷入 | 可能减少合作 |
| 外交施压 | 评估是否让步 | 观望 | 评估瓦希德实力 |
| 封锁反制成功 | 考虑升级 | 可能转向瓦希德 | 增加合作意愿 |
| 经济示好 | 不为所动 | 可能改善关系 | 增加合作意愿 |

### AI学习机制（有限记忆）

```
AI记录玩家最近5回合的行动模式：
- "激进型"玩家（频繁军事行动）：AI倾向更早升级封锁
- "防守型"玩家（偏向外交）：AI倾向更耐心等待
- "反复横跳"玩家（摇摆不定）：AI困惑，决策随机性+30%

限制：
- 学习不影响客观利益计算
- 超过5回合的模式逐渐淡忘
- 最多影响决策权重±20%
```

### AI协调行动机制

```
协调由黄金领发起（需消耗影响力）：
1. 黄金领提出协调提议（如联合封锁）
2. 相关AI评估是否配合（基于自身利益）
3. 配合给予奖励（分摊代价/获得补偿）
4. 不配合则不参与（但也不破坏）

协调效果：
- 联合封锁：效果 ≈ 各方单独封锁 × 1.5（协调成本后）
- 联合军事行动：成功率提升
- 信息共享：配合方获得情报折扣

协调失败：
- 某方拒绝配合：协调破裂
- 协调破裂后：该方单独行动（反而更不可预测）
```

### AI态度选项（替代难度等级）

```
玩家可选择的AI态度（不影响AI基础能力）：

正常AI（默认）：
- 按设计的人格和权重决策

保守AI（适合新手）：
- 风险承受降低
- 决策权重：风险因素 -10%
- 更倾向维持现状

激进AI（增加挑战）：
- 风险承受提高
- 决策权重：风险因素 +10%
- 更倾向主动行动
```

## Formulas 公式

### 核心计算

#### 公式1：行动得分计算
```
行动得分 = Σ(因素_i × 权重_i) + random(-5, +5)

因素权重由人格模块定义
random项保证不可完全预测性
```

#### 公式2：黄金领封锁升级决策
```
封锁升级得分 = 
    (瓦希德威胁程度 × 0.4)
  + (升级封锁代价 × 0.3)
  + (国内政治倾向 × 0.2)
  + (外交谈判可能性 × 0.1)
  + random(-5, +5)

瓦希德威胁程度：
- 火海咽喉被威胁：+40
- 瓦希德高原被加固：+20
- 灰烬众扩张：+15
- 正常状态：0

升级封锁代价：
- 盟友配合意愿高（>70%）：0
- 盟友配合意愿中（40-70%）：+10
- 盟友配合意愿低（<40%）：+20
- 全球油价高位：+15（升级导致油价跌）

国内政治倾向：
- 强硬派主导：-15（倾向升级）
- 温和派主导：+15（倾向克制）

外交谈判可能性：
- 瓦希德释放谈判信号：+25（倾向暂缓）
- 谈判可能性低：0
```

#### 公式3：金帐合众能源政策
```
能源政策得分 = 
    (油价目标 × 0.4)
  + (市场份额 × 0.3)
  + (外交关系 × 0.2)
  + (国内需求 × 0.1)
  + random(-3, +3)

油价目标（基准）：
- 希望油价上涨：增产（-15分）
- 希望油价稳定：维持（0分）
- 希望油价下跌：减产（+15分）

市场份额：
- 市场份额下降：减产（+10分）
- 市场份额稳定：维持（0分）
- 市场份额上升：增产（-10分）

外交关系：
- 与黄金领关系好：配合增产（-10分）
- 与黄金领关系差：减产要价（+10分）
```

#### 公式4：北境共主武器销售配额
```
武器销售配额 = 基础配额 × (关系系数) × (风险系数) × (库存系数)

基础配额 = 3（每回合最大交易次数）

关系系数：
- 同盟(>+80)：1.3
- 友好(+41~+80)：1.0
- 一般(-40~+40)：0.8
- 冷淡(-80~-41)：0.5
- 敌对(<-80)：0.2

风险系数：
- 黄金领未施压：1.0
- 黄金领轻度施压：0.7
- 黄金领重度施压：0.3

库存系数：
- 库存充足：1.0
- 库存紧张：0.6
```

## Edge Cases 边界情况

| 边界情况 | 处理方式 |
|---------|---------|
| AI决策导致自身利益受损 | 人格权重会自动调整（痛苦学习） |
| 多势力同时决策产生冲突 | 按势力优先级顺序执行（黄金领>圣火序>金帐>北境>东土>灰烬众） |
| AI与AI之间冲突 | 根据人格计算谁让步；若势均力敌则各退一步 |
| 局势发生剧变（玩家核打击） | AI立即进入"紧急状态"，重新评估所有决策 |
| 玩家长时间不行动 | AI逐渐"忘记"玩家威胁，回归默认策略 |

## Data Interfaces 数据接口

### G向其他系统提供的接口

```typescript
interface FactionAISystem {
    // 获取某势力的决策
    getFactionDecision(factionId: FactionId): FactionDecision
    
    // 获取所有势力的决策列表
    getAllDecisions(): FactionDecisionMap
    
    // 评估某行动对特定势力的影响
    assessActionForFaction(
        action: PlayerAction,
        factionId: FactionId
    ): ImpactAssessment
    
    // 获取势力的当前人格状态
    getFactionPersonality(factionId: FactionId): PersonalityState
}

interface FactionDecision {
    factionId: FactionId
    turnNumber: number
    decisionType: 'military' | 'diplomatic' | 'blockade' | 'trade' | 'posture'
    selectedAction: Action
    actionScore: number
    alternatives: Action[]
    reasoning: string
}
```

## Tuning Knobs 调参旋钮

| 旋钮 | 默认值 | 可调范围 | 调整后果 |
|------|--------|---------|---------|
| AI决策随机性 | ±5 | ±3~±8 | 影响AI可预测程度 |
| 黄金领强硬派比例 | 50% | 30%-70% | 影响封锁升级倾向 |
| AI响应延迟（回合） | 1回合 | 0-2回合 | 影响玩家先手优势 |
| 紧急响应阈值 | 咽喉危机 | — | 影响AI紧急反应速度 |
| 北境共主多边下注系数 | 0.8 | 0.6-1.0 | 影响北境立场稳定性 |
| 金帐合众机会主义系数 | 0.7 | 0.5-0.9 | 影响金帐立场稳定性 |
| AI学习机制模式记录数 | 5回合 | 3-7回合 | 影响AI对玩家行为模式的记忆 |
| AI学习权重影响 | ±20% | 15%-25% | 影响AI根据模式调整决策的程度 |
| AI协调成本系数 | 0.5 | 0.3-0.7 | 影响联合行动的协调代价 |

**默认值已确定，待Alpha阶段平衡测试验证。**
