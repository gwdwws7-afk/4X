# E 内部政治系统 (Internal Politics System)

> **状态**: Draft
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P1（内部稳定核心）

## Overview 概述

内部政治系统定义瓦希德内部的派系斗争、政治稳定度和路线分歧。玩家需要在不同派系之间平衡，以确保内部稳定。

## Player Fantasy 玩家幻想

玩家感受到"最大的敌人往往在内部"。

激进派在喊打喊杀，要求对黄金领采取更激进的行动。温和派在呼吁克制，担心过度挑衅会引来更大的灾难。务实派在说，先活下去再说，别管什么路线了。每一个决策都在影响派系平衡——激进派太高兴了，温和派就会失望；温和派满意了，激进派可能闹事。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 派系定义

| 派系 | 英文 | 核心诉求 | 影响力度 |
|------|------|---------|---------|
| **激进派** | Hawks | 对黄金领强硬，优先军事路线 | 30% |
| **温和派** | Moderates | 外交解决，优先和平路线 | 30% |
| **务实派** | Pragmatists | 生存第一，优先经济路线 | 40% |

#### 派系满意度

```
派系满意度 = 100 - Σ(不满因素) + Σ(满意因素)

派系满意度范围：0-100

不满因素：
- 军事失败：-10~-20
- 经济困难：-5~-15
- 外交让步：-5~-10
- 意识形态受挫：-5~-10

满意因素：
- 军事胜利：+10~+20
- 经济改善：+5~+15
- 外交成功：+5~+10
- 意识形态胜利：+5~+10
```

#### 派系冲突

```
派系冲突触发条件：
- 某派系满意度 < 30：派系开始不满
- 某派系满意度 < 15：派系冲突
- 派系冲突持续3回合：派系危机

派系冲突效果：
- 相关军事/外交行动成本+30%
- 社稷值每回合-3
- 路线分歧事件触发
```

#### 内部稳定路线

```
内部稳定路线 = 派系平衡

路线A：军事优先（激进派主导）
- 军事行动成本-15%，成功率+5%
- 外交难度+20%
- 激进派满意度+10，温和派-5

路线B：外交优先（温和派主导）
- 外交成本-15%，成功率+5%
- 军事行动成本+20%
- 温和派满意度+10，激进派-5

路线C：经济优先（务实派主导）
- 经济效率+15%
- 军事和外交成本+10%
- 务实派满意度+10，其他派系-3
```

### 社稷值与内部政治

```
社稷值 = 派系平衡 × 经济基础 × 外部压力

社稷值计算：
= (基础值70) × 派系系数 × 经济系数 × 外部压力系数

派系系数：
- 派系冲突：×0.7
- 派系平衡：×1.0
- 派系和谐：×1.2

经济系数：
- 经济健康度>70：×1.1
- 经济健康度40-70：×1.0
- 经济健康度<40：×0.8

外部压力系数：
- 封锁压力高：×0.9
- 军事威胁高：×0.85
- 外部援助：×1.1
```
