# D2 军事-政治联动 (Military-Political Linkage System)

> **状态**: Draft
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P2（军事政治联动）

## Overview 概述

军事-政治联动系统定义军事行动如何影响内部政治（D2→E），以及内部政治如何影响军事行动可用性（E→D1）。

## Player Fantasy 玩家幻想

玩家感受到"军事胜利不是终点，政治消化才是"。

你打下了贝鲁特，但这只是开始。接下来你需要派驻官员维持秩序，安抚当地民众，处理难民问题。如果处理不好，占领区的稳定度会持续下降，最终爆发叛乱，你不但保不住贝鲁特，还可能赔上整个地区的控制权。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 军事行动的政治代价

| 军事行动 | 社稷值代价 | 派系满意度变化 |
|---------|-----------|---------------|
| 代理人行动 | -2 | 激进派+2，温和派-1 |
| 特种部队 | -3 | 激进派+3，温和派-2 |
| 咽喉威胁 | -5 | 激进派+5，温和派-3 |
| 不对称防御 | -4 | 温和派+3，激进派-1 |
| 核威慑展示 | -10 | 激进派+10，温和派-5 |
| 全面战争 | -20 | 全体派系不满 |

#### 军事行动的政治收益

| 结果 | 社稷值收益 | 派系满意度变化 |
|------|-----------|---------------|
| 节点控制 | +3 | 全体派系+2 |
| 军事胜利（大规模） | +5 | 激进派+5，温和派+3 |
| 保卫成功 | +4 | 温和派+4，激进派+2 |

#### 政治消化机制

```
占领节点后政治消化：
每回合需投入：
- 社稷值-2（维持占领）
- 战械-1（驻军）

消化周期：
- 小节点：3回合
- 中节点：5回合
- 大节点（咽喉/能源）：8回合

消化完成前：
- 节点稳定度持续下降
- 可能爆发叛乱（D3代理人民事系统）
```

### 军事行动与内部政治联动

```
E → D1 联动（内部政治影响军事）：
- 温和派主导：军事行动成本+20%，成功率+10%
- 激进派主导：军事行动成本-10%，成功率-5%
- 派系冲突：部分行动不可用

D2 → E 联动（军事影响内部）：
- 军事胜利：派系满意度提升
- 军事失败：派系冲突加剧
- 占领新节点：需要政治消化资源
```
