# B1 星币-能源金融系统 (Star Currency-Energy Finance System)

> **状态**: Designed
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P0（不对称博弈核心）, P1（代价结算基础）

## Overview 概述

星币-能源金融系统是游戏的经济命脉，定义能源产出的国际结算方式、封锁状态下的出口收入计算、以及绕过星币结算的替代渠道。

瓦希德帝国的核心困境在于：能源产量再高，被踢出星币结算体系后也无法转化为国际购买力。本系统将"能源"和"金钱"解耦——玩家有能源不等于有钱，有钱也不一定能买到你需要的东西。这种错位是经济系统的核心张力来源。

## Player Fantasy 玩家幻想

玩家感受到"有钱买不到"和"有货卖不出"的双重窒息。

手上攥着满满的火油，但星币储备见底——国际军火商的报价只能用星币结算，而你的星币来源因为封锁几乎断绝。好不容易找到北境共主愿意以货易货，但他们开出的价格让这比交易几乎无利可图。东土商盟倒是愿意收你的能源，但他们的"商盟券"只能在特定渠道使用，买不了最先进的武器。

这就是瓦希德帝国的真实处境：资源丰富但金融孤立。玩家每回合都在计算：我的能源能换多少星币？星币不够的话商盟券够不够？商盟券能买什么？这一系列决策就是本系统的核心体验。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 星币结算体系架构

```
全球能源贸易结算体系
│
├── 星币结算层（黄金领霸权）
│   ├── 所有主流能源贸易以星币结算
│   ├── SWIFT金融网络控制
│   └── 黄金领控制结算通道
│
├── 替代结算层
│   ├── 商盟券结算（东土商盟渠道）
│   │   ├── 瓦希德 ↔ 东土商盟 直接结算
│   │   └── 适用范围受限（特定商品类别）
│   ├── 北境本币结算（北境共主渠道）
│   │   ├── 能源换武器/技术
│   │   └── 汇率不利于瓦希德
│   └── 以物易物（紧急渠道）
│       ├── 能源 ↔ 商品/武器 直接交换
│       └── 效率最低但最难被封锁
│
└── 灰色市场层
    ├── 中立国转口贸易
    ├── 灰色市场税率（成本加成）
    └── 数量有限且不稳定
```

#### 结算渠道优先级

当瓦希德需要购买国际商品时，按以下优先级使用结算方式：

| 优先级 | 结算渠道 | 可购买范围 | 汇率/成本 | 可用性 |
|--------|---------|-----------|----------|--------|
| 1 | 星币储备（金叶） | 全品类 | 正常汇率 | 受封锁影响，可获得量少 |
| 2 | 商盟券 | 东土商盟限定商品 | 1:1（东土内部） | 需要贸易协议 |
| 3 | 北境本币结算 | 北境共主商品 | 不利于瓦希德（1.3倍） | 外交关系好时可用 |
| 4 | 以物易物 | 特定商品 | 效率70% | 需双方同意 |
| 5 | 灰色市场 | 受限商品 | 成本+40% | 不稳定，数量有限 |

#### 封锁状态定义

| 封锁等级 | 描述 | 星币获取限制 | 能源出口限制 |
|----------|------|-------------|-------------|
| **无封锁** | 正常国际贸易 | 100%可获得 | 100%正常收入 |
| **金融封锁** | 金流封锁生效（SWIFT切断） | 20%可获得 | 正常 |
| **能源禁运** | 出口禁运生效 | 20%可获得 | 20%正常收入 |
| **全面封锁** | 金融+能源双重封锁 | 10%可获得 | 10%正常收入 |
| **最高封锁** | 叠加海上封锁/二级封锁 | 5%可获得 | 5%正常收入 |

封锁状态由B2封锁系统控制，B1负责在对应封锁状态下计算经济数据。

#### 能源出口收入计算

**公式**：
```
能源收入 = 国内能源消耗收入 + 国际能源出口收入

国内能源消耗收入 = 国内能源产量 × 国内消费系数 × 国内能源价格
国内消费系数 = 0.3（30%能源用于国内，70%可出口）
国内能源价格 = 基准价格 × (全球油价波动系数)

国际能源出口收入 = 出口能源量 × 出口价格 × 星币可得系数 × 封锁折扣

出口能源量 = 国内能源产量 × 出口比例 × 节点畅通系数
出口比例 = 0.7（70%用于出口，30%国内消耗）
出口价格 = 基准价格 × (全球油价波动系数) × (1 + 瓦希德市场份额加成)
星币可得系数 = f(封锁等级)  // 见上表
封锁折扣 = f(封锁等级)  // 见上表
```

**基准参数**：
- 基准价格 = 80金叶/单位火油
- 全球油价波动系数 = 0.8 ~ 1.4（每回合随机波动，受金帐合众产能影响）
- 瓦希德市场份额加成 = -0.1（市场份额低，定价权弱）

**计算示例**：
```
场景：瓦希德高原产8火油，无封锁
国内收入 = 8 × 0.3 × 80 = 192金叶
出口收入 = 8 × 0.7 × 80 × 1.0 × 1.0 = 448金叶
总能源收入 = 640金叶/回合

场景：同上，但处于能源禁运封锁
出口收入 = 8 × 0.7 × 80 × 0.2 × 0.2 = 44.8金叶
总能源收入 = 192 + 44.8 = 236.8金叶/回合
```

#### 全球油价机制

| 金帐合众行动 | 油价影响 | 对瓦希德的影响 |
|-------------|---------|---------------|
| 增加产能 | -15% | 出口收入减少 |
| 减少产能 | +20% | 出口收入增加 |
| 与瓦希德达成能源协议 | +10% | 出口收入增加 |

全球油价每回合根据金帐合众产能决策和市场需求波动。

### 替代结算渠道详解

#### 商盟券渠道

```
商盟券获取：
- 与东土商盟签订贸易协议后解锁
- 每签订1份贸易协议，商盟券上限+10
- 能源出口至东土商盟，换取商盟券
- 兑换汇率：1火油 = 70商盟券（低于星币价格，因为东土承担了封锁风险）
```

```
商盟券使用范围：
- 购买东土商盟商品（工业品、技术、部分武器）
- 不可购买：最先进武器、核技术、某些战略物资
- 有效期内必须使用（每10回合过期50%）
```

#### 北境本币结算渠道

```
北境本币结算：
- 需要与北境共主有"有限同盟"以上关系
- 以能源换北境武器/技术
- 汇率：1火油 = 60北境银（低于市价，因为北境承担政治风险）
- 数量上限：每回合最多3次大宗交易（受外交关系限制）
```

#### 以物易物渠道

```
以物易物机制：
- 任何情况下可尝试
- 需要双方同意（AI接受率受外交关系影响）
- 效率折扣：70%（能源换商品）或 60%（能源换武器）
- 无法进行大规模交易（每次最多2单位战械等价物）
```

#### 灰色市场

```
灰色市场：
- 通过中立国（瑞士、新加坡等）转口
- 税率：成本+40%
- 数量不稳定（每回合1-3次机会）
- 被黄金领发现风险：每次交易10%概率触发"走私曝光"事件
- 走私曝光累计惩罚上限：-30（达到后渠道永久关闭）
- 豁免条件：与北境共主有"同盟"以上关系时，走私至北境不触发惩罚
```

### 购买力转换表

| 瓦希德资源 | 可兑换 | 实际购买力 | 说明 |
|-----------|--------|-----------|------|
| 1火油 → 星币 | 80金叶 | 80购买力 | 无折扣，但获取受限 |
| 1火油 → 商盟券 | 70商盟券 | 70购买力 | 适用范围受限 |
| 1火油 → 北境银 | 60北境银 | 60购买力（换武器时80） | 武器刚需时有用 |
| 1火油 → 以物易物 | ~50等价物 | 50购买力 | 效率最低 |
| 1火油 → 灰色市场 | ~45等价物 | 45购买力 | 还要承担被查获风险 |

### 星币的消耗用途

| 消耗类别 | 消耗方式 | 说明 |
|---------|---------|------|
| **军事采购** | 金叶直接购买战械 | 标准价格：10金叶/战械 |
| **外交筹码** | 金叶购买外交协议加成 | 视协议类型定价 |
| **科技研发** | 金叶加速研发进度 | 基础价格按研发项目定价 |
| **情报行动** | 金叶购买侦察信息 | 高价值情报价格昂贵 |
| **紧急购置** | 金叶从任意渠道购买 | 价格×1.2，不限渠道 |

### 结算流程

每个回合结算时，B1系统的计算顺序：

```
回合结算 - B1经济计算
│
├── 1. 计算封锁状态
│   └── 读取B2系统当前封锁等级
│
├── 2. 计算全球油价
│   └── 读取金帐合众产能决策 + 市场波动
│
├── 3. 计算能源收入
│   ├── 读取H2各能源节点产出
│   ├── 计算国内消耗收入
│   ├── 计算国际出口收入（考虑封锁）
│   └── 输出：火油 → 金叶
│
├── 4. 结算替代渠道
│   ├── 商盟券：检查贸易协议，计算可兑换量
│   ├── 北境结算：检查外交关系，计算配额
│   └── 灰色市场：随机决定可用次数
│
├── 5. 更新各货币储备
│   ├── 金叶 += 能源收入 - 军事采购 - 外交支出
│   ├── 商盟券 += 贸易收入 - 定向采购
│   └── 北境银 += 能源易货 - 北境采购
│
└── 6. 输出经济状态快照
    └── 供B5经济结算系统汇总
```

## Formulas 公式

### 核心计算

#### 公式1：能源出口收入
```
国际出口收入 = 
    (能源产量 × 出口比例 × 出口价格 × 节点畅通系数) 
    × 星币可得系数(封锁等级) 
    × 封锁折扣(封锁等级)

出口比例 = 0.7
节点畅通系数 = 
    咽喉要道畅通: 1.0
    咽喉要道受压: 0.7
    咽喉要道中断: 0.0
```

#### 公式2：全球油价波动
```
当回合油价 = 基准油价 × (1 + 金帐产能调整 + 市场微调)

基准油价 = 80金叶/单位
金帐产能调整 = 
    增产: -0.15（每增产一档）
    减产: +0.20（每减产一档）
    上限约束: -0.30 ~ +0.40（防止极端值）
市场微调 = random(-0.05, +0.05)（保持适度不可预测）
```

#### 公式3：商盟券兑换
```
商盟券获得 = 出口至东土能源量 × 70商盟券/火油
商盟券使用过期 = 当前商盟券 × 0.5（每10回合一次）
```

#### 公式4：北境结算汇率
```
北境银获得 = 出口至北境能源量 × 60北境银/火油
武器购买力 = 北境银 × 1.33（北境武器以低价计算）
```

#### 公式5：灰色市场成本
```
灰色市场价格 = 标准价格 × 1.4
走私曝光率 = 10%（每次交易）
```

## Edge Cases 边界情况

| 边界情况 | 处理方式 |
|---------|---------|
| 能源节点被切断但有替代路线 | 替代路线效率-40%，节点畅通系数降至0.6 |
| 星币储备耗尽 | 只能使用替代渠道；军事采购受限 |
| 商盟券过期 | 每10回合：未使用的商盟券×50%保留，最多过期当前储备的50%；UI提前3回合提示 |
| 灰色市场走私曝光 | 触发外交事件：与曝光国家关系下降，可能触发额外封锁 |
| 全球油价暴跌 | 能源出口收入锐减，可能低于国内消耗收入 |
| 多渠道同时可用 | 按优先级自动结算（可手动调整） |

## Data Interfaces 数据接口

### B1向其他系统提供的接口

```typescript
interface FinanceSystem {
    // 查询当前经济状态
    getEconomicState(): EconomicState
    
    // 查询封锁状态下某资源的实际购买力
    getEffectivePurchasingPower(resource: ResourceType): number
    
    // 计算指定数量的能源可兑换的各渠道货币量
    calculateEnergyExchange(energyAmount: number): ExchangeResult
    
    // 查询各结算渠道可用性
    getChannelAvailability(): ChannelAvailability
    
    // 查询全球油价
    getGlobalOilPrice(): number
}

interface EconomicState {
    goldLeaves: number              // 金叶储备
    fireOil: number                 // 火油储备
    tradeNotes: number              // 商盟券储备
    northCoins: number              // 北境银储备
    blockadeLevel: BlockadeLevel    // 当前封锁等级
    globalOilPrice: number          // 当前全球油价
    energyExportIncome: number       // 本回合能源出口收入
    domesticEnergyIncome: number     // 本回合国内能源收入
}
```
