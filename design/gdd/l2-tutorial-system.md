# L2 新手教程系统 (Tutorial System)

> **状态**: Draft
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P3（新用户体验）

## Overview 概述

新手教程系统引导新玩家了解游戏机制，包括基础教学和进阶提示。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 教程阶段

| 阶段 | 内容 | 时机 | 时长 |
|------|------|------|------|
| **入门** | 游戏目标、界面介绍、回合概念 | 第一回合 | 5分钟 |
| **资源** | 7种资源介绍、经济循环 | 第一回合内政阶段 | 3分钟 |
| **军事** | 军事行动基础、不对称战争 | 第二回合军事阶段 | 5分钟 |
| **外交** | 外交关系、协议签订 | 第三回合外交阶段 | 5分钟 |
| **胜利** | 四种胜利路径介绍 | 第三回合结束 | 3分钟 |

#### 教程提示类型

| 提示类型 | 显示方式 | 示例 |
|---------|---------|------|
| **气泡提示** | 点击UI元素时显示 | "这是资源栏，显示当前火油储备" |
| **高亮标注** | 目标区域发光 | 标注"这是你的第一个行动点" |
| **行动建议** | 推荐行动带边框 | 推荐"先进行内政改革" |
| **跳过选项** | 教程角落 | "跳过教程"按钮 |

#### 教程流程

```
教程开始：
1. 显示"欢迎来到抵抗轴心"
2. 介绍游戏目标（打破封锁/生存）
3. 高亮界面各区域

阶段1：内政阶段
1. 提示行动点概念
2. 引导首次内政行动
3. 解释资源分配

阶段2-5：重复类似流程

教程结束：
1. 显示"教程完成"
2. 提供"再来一次教程"选项
3. 开启正常游戏
```

### 进阶提示

```
进阶提示（非教程模式）：
- 玩家首次执行某类行动时显示提示
- 如首次封锁升级：显示"封锁升级会影响..."

提示冷却：
- 每个提示类型只显示一次
- 可在设置中重置提示
```
