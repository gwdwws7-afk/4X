# L4 本地化系统 (Localization System)

> **状态**: Draft
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P4（国际化支持）

## Overview 概述

本地化系统支持游戏多语言版本，包括文字翻译、语音（可选）、本地化格式等。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 支持语言

| 语言 | 代码 | 状态 |
|------|------|------|
| 中文（简体） | zh-CN | 默认 |
| 中文（繁体） | zh-TW | 支持 |
| 英语 | en-US | 支持 |
| 波斯语 | fa-IR | 未来支持 |
| 阿拉伯语 | ar-SA | 未来支持 |
| 俄语 | ru-RU | 未来支持 |

#### 本地化内容

```
本地化内容类型：

1. 界面文字
   - 菜单、按钮、标签
   - 资源名称、状态描述

2. 叙事文字
   - 事件标题和描述
   - 胜利/失败叙事
   - 教程引导

3. 数值格式
   - 数字格式（1,000 vs 1.000）
   - 日期格式（2028 H2）
   - 货币格式

4. 音频文字
   - 语音对话字幕
   - 语音提示文字
```

#### 本地化架构

```
本地化架构：

游戏资源
    ↓
本地化加载器
    ↓
当前语言字符串表
    ↓
界面显示

字符串表格式：JSON/CSV
{
    "ui.main_menu.start": "开始游戏",
    "ui.main_menu.load": "加载游戏",
    "event.gulf_crisis.title": "火海咽喉危机"
}
```

#### 运行时语言切换

```
语言切换：
- 设置中选择语言
- 无需重启游戏
- 即时切换

热加载：
- 语言切换时重新加载字符串表
- 当前界面立即更新
- 不中断游戏
```
