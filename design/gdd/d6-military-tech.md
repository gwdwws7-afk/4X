# D6 军事科技系统 (Military Technology System)

> **状态**: Draft
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P2（长期成长系统）

## Overview 概述

军事科技系统是玩家的"长期成长"机制。通过科技研发，瓦希德可以提升军事能力、获得新装备、加强防御。

本系统的核心设计目标是：让科技成为值得投入的方向，但不是"科技万能"。研发需要时间和资源，而且对手也会进步。

## Player Fantasy 玩家幻想

玩家感受到"科技投资是长期赌博"。

研发一项新技术需要5回合、100金叶——这段时间内你的资源都被锁定了。研发完成后，你获得的装备优势可能在3回合内被敌方科技反超。但如果你不研发，对手的科技迟早会碾压你。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 科技树结构

```
军事科技树（简化版）：

[基础军事]
    ├── [防御强化I] → [防御强化II] → [防御强化III]
    ├── [攻击强化I] → [攻击强化II] → [攻击强化III]
    └── [不对称战术I] → [不对称战术II]

特殊科技（需协议解锁）：
├── [北境武器I]：需要军事技术合作L1
├── [北境武器II]：需要军事技术合作L2
└── [北境武器III]：需要军事技术合作L3
```

#### 科技效果

| 科技 | 效果 | 研发成本 | 研发周期 |
|------|------|---------|---------|
| 防御强化I | 不对称防御成功率+5% | 50金叶 | 3回合 |
| 防御强化II | 不对称防御成功率+5% | 80金叶 | 5回合 |
| 防御强化III | 不对称防御成功率+5%，地形加成+10% | 120金叶 | 7回合 |
| 攻击强化I | 代理人行动成功率+5% | 50金叶 | 3回合 |
| 攻击强化II | 代理人行动成功率+5% | 80金叶 | 5回合 |
| 攻击强化III | 代理人行动成功率+5%，特种部队+10% | 120金叶 | 7回合 |
| 不对称战术I | 咽喉威胁成功率+10% | 60金叶 | 4回合 |
| 不对称战术II | 咽喉威胁成功率+10%，封锁升级风险-10% | 100金叶 | 6回合 |
| 北境武器I | 解锁北境L1装备，攻击+10% | 70金叶 | 4回合 |
| 北境武器II | 解锁北境L2装备，攻击+15% | 110金叶 | 6回合 |
| 北境武器III | 解锁北境L3装备，攻击+20%，防御+10% | 150金叶 | 8回合 |

#### 研发规则

```
研发同时只能进行1项
研发进度 = 每周投入资金 / 总成本
研发可取消（已投入资金不返还）
研发完成自动装备

研发中断：
- 若资源不足继续投入，研发暂停（不取消）
- 暂停超过3回合，研发自动取消
```

### 科技与游戏机制联动

```
科技 → D1军事行动：
- 科技提供行动成功率加成
- 永久生效，无消耗

科技 → B1经济：
- 研发消耗金叶
- 部分科技解锁特殊交易渠道

科技 → 北境协议联动：
- 北境武器I/II/III需要对应级别的军事技术合作协议
```
