# D4 核威慑系统 (Nuclear Deterrence System)

> **状态**: Draft
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P2（终极威慑机制）

## Overview 概述

核威慑系统定义核武器作为威慑工具而非实际武器的使用机制。核打击是禁区，但核威慑是真实存在的策略工具。

## Player Fantasy 玩家幻想

玩家感受到"核武器是挂在所有人头上的剑"。

你展示了核威慑能力，这让黄金领在考虑全面战争时更加谨慎。但每次展示核威慑，你的国际声望都会下降——世界不喜欢被核武器威胁的国家。你永远不会真正使用核武器（那意味着全面毁灭），但你可以通过展示威慑来改变博弈规则。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 核威慑状态

```
核威慑状态 = {
    瓦希德核能力: '无' | '有限' | '可信' | '强化' | '绝对'
    核弹头数量: 0-100
    核威慑展示冷却: 回合数
}
```

#### 核威慑等级

| 等级 | 核弹头 | 威慑效果 | 展示代价 |
|------|--------|---------|---------|
| 无 | 0 | 无威慑 | — |
| 有限 | 1-10 | 基础威慑 | 灰烬志-10 |
| 可信 | 11-30 | 全面战争威慑 | 灰烬志-15，朝贡序-10 |
| 强化 | 31-60 | 附加封锁突破能力 | 灰烬志-20，朝贡序-15 |
| 绝对 | 61+ | 全面绝对威慑 | 灰烬志-25，朝贡序-20 |

#### 核威慑展示效果

```
D1核威慑展示触发后：
1. 敌方全面战争选项被锁定2回合
2. 封锁升级审议暂缓
3. 敌方军事行动成功率-20%
4. 全球反应：与所有国家关系-5
5. 瓦希德国际声望受损
```

#### 核打击禁区

```
核打击触发条件：无（绝对禁区）

若触发（游戏Bug或极端情况）：
- 游戏立即结束
- 触发"相互毁灭"终局
- 所有玩家存档无法继续
```

#### 核威慑与外交

```
核威慑对外交的影响：
- 威慑展示期间：与所有国家外交难度+20%
- 核能力强化后：与北境共主关系+10（他们欣赏核威慑）
- 核能力弱化后：与黄金领关系+15（威胁降低）
```
