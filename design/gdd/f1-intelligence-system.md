# F1 情报系统 (Intelligence System)

> **状态**: Draft
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P2（信息不对称核心）

## Overview 概述

情报系统是游戏"信息不对称"设计的核心。玩家（瓦希德）永远不知道敌人的全部计划，但可以通过侦察、渗透、反情报等手段逐步揭示真相。

本系统的核心设计目标是：让信息本身成为稀缺资源。知道得越多，消耗越大；而且知道的不一定是真的。

## Player Fantasy 玩家幻想

玩家感受到"情报是生命线，但情报也可能欺骗你"。

你花了大价钱侦察黄金领在火海咽喉的军事部署，发现他们的舰队比预期少——这是真的吗？还是对方故意让你看到假的？你派渗透者进入圣火序内部，得到的情报说他们即将发动攻击——但消息来源可靠吗？

情报的价值不只是"知道"，而是"知道该相信什么"。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 情报行动类型

| 行动类型 | 英文 | 行动点 | 消耗 | 效果 |
|---------|------|--------|------|------|
| **侦察** | Reconnaissance | 1 | 战械-1 | 揭示指定节点的迷雾（降低1级） |
| **深度侦察** | Deep Recon | 2 | 战械-2 | 揭示节点归属和详细数值 |
| **渗透** | Infiltration | 2 | 战械-1，朝贡序-3 | 获得敌方意图情报（揭示AI决策倾向） |
| **反情报** | Counter-Intelligence | 1 | 战械-1 | 保护己方节点（迷雾提升1级） |
| **心理战** | Psychological Warfare | 1 | 灰烬志-5 | 向敌方传播假情报（影响AI决策） |

#### 情报等级

```
情报等级（从低到高）：

[无情报] → [迹象] → [碎片] → [完整] → [完全掌握]

无情报：迷雾全黑，什么都看不到
迹象：只能看到节点类型（能源/咽喉/贸易/意识形态）
碎片：可以看到归属，看不到详细数值
完整：可以看到归属、产出、稳定度
完全掌握：完整 + 可以看到争夺点数和历史
```

#### 迷雾等级与情报等级对应

| 迷雾等级 | 显示效果 | 对应情报等级 |
|---------|---------|------------|
| 3级（全雾） | 完全黑 | 无情报 |
| 2级 | 只能看到节点类型 | 迹象 |
| 1级 | 可以看到归属 | 碎片 |
| 0级（无雾） | 完全可见 | 完整/完全掌握 |

#### 情报可靠性

```
情报可靠性：50% - 95%（取决于情报来源质量）

可靠性计算：
基础可靠性 = 60%
来源加成：
- 普通侦察：+10%
- 深度侦察：+20%
- 渗透（高价值目标）：+5%
- 渗透（低价值目标）：+15%

目标反击：
- 敌方反情报能力：-5%~-20%
- 目标为咽喉要道：-15%（高价值目标反情报强）

最终可靠性 = 基础可靠性 + 来源加成 - 目标反击

情报真实性判定：
- 可靠性 > 80%：情报几乎肯定真实
- 可靠性 60-80%：情报大部分真实，可能有误导
- 可靠性 < 60%：情报可能是假的，需要验证
```

#### 情报行动效果

**侦察**：
```
效果：指定节点迷雾等级 -1
目标限制：每次最多侦察2个节点
冷却：无
成功概率：80%（基础）
```

**深度侦察**：
```
效果：指定节点迷雾等级 -2（降至0级）
目标限制：每次最多1个节点
冷却：3回合（同一节点）
成功概率：60%
附加效果：揭示节点详细数值
```

**渗透**：
```
效果：揭示敌方意图（AI决策倾向）
目标限制：每次1个势力
情报类型：
- 敌方当前封锁计划
- 敌方军事行动计划
- 敌方外交意图
成功概率：50%
可靠性：基础60%（可能存在误导）
```

**反情报**：
```
效果：己方节点迷雾等级 +1（更难被侦察）
目标限制：每次最多2个节点
持续时间：5回合
成功概率：70%
```

**心理战**：
```
效果：向敌方传播假情报
目标：指定敌方势力
假情报类型：
- 虚假军事部署（影响AI决策）
- 虚假外交意图（影响AI关系评估）
成功概率：40%
持续时间：3回合
反制：敌方渗透成功则假情报被识破
```

### 情报与游戏机制联动

#### 情报-D1联动

```
侦察揭示的节点信息影响D1军事行动：
- 完整情报：军事行动成功率 +15%
- 碎片情报：成功率不变
- 无情报：成功率 -10%
```

#### 情报-G联动

```
渗透揭示的AI意图影响G框架：
- 敌方封锁计划：玩家可提前应对
- 敌方军事行动：玩家可优先防御
- 敌方外交意图：玩家可抢先谈判
```

#### 情报-B2联动

```
反情报影响B2封锁效果：
- 敌方对瓦希德的封锁评估准确度下降
- 封锁升级概率：-5%~-15%
```

## Formulas 公式

### 核心计算

#### 公式1：情报可靠性
```
情报可靠性 = 60% + 来源加成 - 目标反击

来源加成：
- 普通侦察：+10%
- 深度侦察：+20%
- 渗透（高价值）：+5%
- 渗透（低价值）：+15%

目标反击：
- 普通节点：-5%
- 重要节点（能源/咽喉）：-15%
- 敌方反情报能力强：-20%
```

#### 公式2：侦察覆盖率
```
回合侦察覆盖率 = (侦察成功次数 / 侦察总次数) × 100%

历史累积（每回合计算）：
- 过去5回合侦察覆盖率 > 70%：AI行为更可预测
- AI行为预测度影响渗透成功率 +10%
```

#### 公式3：反情报有效性
```
反情报有效性 = 基础有效性 × 敌方反情报能力系数

基础有效性 = 70%
敌方反情报能力系数：
- 黄金领：0.8（反情报能力强）
- 圣火序：0.9
- 其他：1.0
```

## Data Interfaces 数据接口

### F1向其他系统提供的接口

```typescript
interface IntelligenceSystem {
    // 执行侦察
    executeRecon(nodeId: string, depth: 'normal' | 'deep'): ReconResult
    
    // 执行渗透
    executeInfiltration(targetFaction: FactionId): InfiltrationResult
    
    // 执行反情报
    executeCounterIntel(nodeId: string): CounterIntelResult
    
    // 执行心理战
    executePsychWar(targetFaction: FactionId, disinformation: DisinfoType): PsychWarResult
    
    // 获取情报报告
    getIntelligenceReport(factionId?: FactionId): IntelligenceReport
}

interface IntelligenceReport {
    factionId: FactionId
    knownIntentions: {
        type: 'blockade' | 'military' | 'diplomatic'
        description: string
        reliability: number
        turnAcquired: number
    }[]
    nodeIntelligence: {
        nodeId: string
        level: IntelligenceLevel
        reliability: number
    }[]
}
```
