# C1 外交关系系统 (Diplomatic Relations System)

> **状态**: Designed
> **作者**: GDC Framework / 游戏设计师
> **最后更新**: 2026-03-31
> **实现支柱**: P1（代价系统核心维度）, P2（叙事驱动）

## Overview 概述

外交关系系统是定义瓦希德与其他势力之间关系好坏的系统。关系不是静态的——每一次军事行动、每一次贸易协议、每一次封锁升级都在改变关系。关系好坏又反过来影响贸易、军事行动效果、封锁强度。

本系统的核心设计目标是：让外交成为"值得经营的资产"，但不是"万能解决方案"。好的外交关系能打开渠道、降低封锁压力；坏的外交关系会让所有事情都更难。

## Player Fantasy 玩家幻想

玩家感受到"外交是四两拨千斤的艺术"。

当你与东土商盟关系友好时，同样的能源出口能得到更多商盟券；当你与北境共主关系紧张时，同样的武器交易要付出更多代价。好的外交关系是隐形的优势——不直接显示在资源栏里，但每一笔交易、每一次行动都在受益。

但外交也有局限。你可以用商盟券贿赂金帐合众的官员，但他们不会因为钱就改变国家政策；你可以与北境共主称兄道弟，但他们随时可能把你卖给黄金领。外交是工具，不是保证。

## Detailed Design 详细设计

### Core Rules 核心规则

#### 势力关系等级

```
关系等级（从低到高）：

[敌对] ←→ [冷淡] ←→ [一般] ←→ [友好] ←→ [同盟]
  -100        -50        0          +50        +100

特殊状态：
- [战争状态]：关系<-80，触发军事行动限制
- [同盟]：关系>+80，可触发联盟防御条款
```

| 关系等级 | 数值范围 | 说明 |
|---------|---------|------|
| 敌对 | -100 ~ -81 | 贸易受限、军事威胁、外交途径极少 |
| 冷淡 | -80 ~ -41 | 有限贸易、无军事合作、外交成本高 |
| 一般 | -40 ~ +40 | 正常贸易、有限外交、关系变化较慢 |
| 友好 | +41 ~ +80 | 优惠贸易、有限合作、外交成本低 |
| 同盟 | +81 ~ +100 | 深度合作、联盟防御条款、共同行动 |

#### 势力外交定位

| 势力 | 基础关系 | 可变化范围 | 外交优先级 | 核心诉求 |
|------|---------|-----------|-----------|---------|
| **黄金领** | -80（敌对） | -100 ~ -60 | 核心对手 | 维持中东霸权、阻止瓦希德拥核 |
| **圣火序** | -70（敌对） | -100 ~ -50 | 主要对手 | 国家安全、区域影响力 |
| **金帐合众** | -20（一般偏冷） | -60 ~ +60 | 中间势力 | 逊尼派领导权、能源财富平衡 |
| **北境共主** | +10（一般偏暖） | -20 ~ +80 | 重要伙伴 | 军事技术出口、南部战略纵深 |
| **东土商盟** | +30（友好） | 0 ~ +90 | 核心伙伴 | 贸易利益、能源进口多样化 |
| **灰烬众** | +60（友好） | +40 ~ +100 | 盟友 | 抵抗叙事、区域影响力 |
| **中立国** | 0（中立） | -30 ~ +50 | 可争取 | 经济利益、战略平衡 |

#### 关系变化机制

**关系每日/回合变化**：

```
关系变化 = Σ(直接变化) + Σ(间接变化) + Σ(事件变化) - 冷却期惩罚

直接变化（玩家行动）：
- 签订贸易协议：+5~+15（取决于协议类型和对象）
- 违反协议：-10~-30
- 军事行动（针对某势力）：-10~-40（取决于规模和目标）
- 军事行动（帮助某势力）：+5~+15
- 外交施压成功：+5~+10
- 外交施压失败：-5~-15

间接变化（系统联动）：
- 封锁升级/解除：相关势力关系±10~±20
- 节点归属变化：相关势力关系±5~±15
- 贸易往来增加：关系缓慢提升（每5回合+1）
- 长期无外交接触：关系缓慢下降（每5回合-1）

事件变化：
- 叙事事件触发（I1系统）
- 全球危机事件
- 自然关系波动

冷却期机制：
- 同一对势力，单次关系变化 > ±20 时，触发3回合冷却
- 冷却期内，相同方向变化效果减半
- 冷却期内，反方向变化不受限制

孤立惩罚（≥3势力关系 < -60时触发）：
- 所有外交行动成本 +20%
- 与中间势力改善关系难度 +30%
- 外交行动成功率 -10%
```

#### 外交阶段行动

外交阶段有**1点行动点**（来自A1回合主循环），另有**2点通用行动点**可分配。

| 外交行动 | 行动点 | 消耗资源 | 效果 | 敌对限制 |
|---------|--------|---------|------|---------|
| **双边谈判** | 1 | 朝贡序-5~10 | 提升目标关系+5~+15 | 敌对时成本×2.5 |
| **多边峰会** | 2 | 朝贡序-15，金叶-30 | 多方关系同时变化；可能达成多方协议 | 需至少一方关系>-40 |
| **外交施压** | 1 | 朝贡序-5 | 目标关系-5~-15 | 敌对时不可用 |
| **协议签订** | 1 | 朝贡序-10 | 签订具体协议；关系提升+协议效果 | 需关系>-40 |
| **断交威胁** | 1 | 朝贡序-5 | 目标关系-10~-20 | 敌对时不可用 |
| **破冰行动** | 2 | 朝贡序-20 | 解锁敌对势力正常外交渠道 | 仅敌对时可用，成本×3 |

#### 外交关系的影响

| 影响维度 | 敌对(-100~-81) | 冷淡(-80~-41) | 一般(-40~+40) | 友好(+41~+80) | 同盟(+81~+100) |
|---------|---------------|--------------|--------------|--------------|----------------|
| **贸易条件** | 完全禁止 | 高价/限量 | 正常价 | 95折 | 90折 |
| **商盟券获取** | 不可用 | 限量 | 正常 | 正常 | 溢价+10% |
| **北境武器** | 不可用 | 限量高价 | 正常 | 正常 | 折扣+15% |
| **外交成本** | — | ×1.5 | ×1.0 | ×0.8 | ×0.6 |
| **军事行动** | 敌对行动升级风险+30 | 敌对行动风险+15 | 正常 | 联合行动可能 | 联盟防御触发 |
| **封锁配合** | 全力封锁 | 积极配合 | 部分配合 | 不配合（自身利益） | 暗中抵制 |

### 外交与游戏机制联动

#### 外交-封锁联动

```
关系影响封锁效果（来自B2）：

当某势力与黄金领关系 < -20：
- 该势力配合二级封锁的意愿降低
- 该势力被施压时更可能"硬顶"
- 东土商盟：配合封锁意愿 80% → 50%
- 北境共主：配合封锁意愿 60% → 30%

当某势力与瓦希德关系 > +60：
- 该势力可能提供"非正式渠道"绕过封锁
- 东土商盟：额外商盟券渠道解锁
- 北境共主：额外武器采购配额
```

#### 外交-军事联动

```
关系影响军事行动（来自D1）：

当目标关系 > +60（友好/同盟）：
- 对该目标的军事行动不可选（除非主动断交）
- 联合军事行动可用（消耗更低，效果更好）

当目标关系 < -80（敌对）：
- 对该目标的军事行动有额外风险加成
- 封锁升级风险+30

当目标为同盟关系（>+80）：
- 若瓦希德被攻击，同盟国可能自动响应（联盟防御条款）
- 联盟防御效果：同盟国在下一回合提供军事/经济援助
```

#### 外交-经济联动

```
关系影响贸易条件：

| 关系等级 | 商盟券获取 | 北境武器价格 | 外交行动成本 |
|---------|-----------|------------|-------------|
| 敌对 | 不可用 | ×2.0 | ×1.5 |
| 冷淡 | 限量(50%) | ×1.5 | ×1.2 |
| 一般 | 正常(100%) | ×1.0 | ×1.0 |
| 友好 | 正常(100%) | ×0.95 | ×0.8 |
| 同盟 | 溢价+10% | ×0.85 | ×0.6 |
```

## Formulas 公式

### 核心计算

#### 公式1：外交行动关系变化
```
关系变化 = 基础变化 + 关系修正 + 情境修正

基础变化：
- 双边谈判成功：+5~+15（取决于朝贡序投入）
- 协议签订：+10~+20（基础+协议价值）
- 外交施压成功：+5~+10
- 外交施压失败：-5~-15
- 军事行动（敌对目标）：-10~-40
- 军事行动（帮助友好目标）：+5~+15

关系修正：
if 当前关系 < 0（敌对/冷淡）:
    关系修正 = ×0.5（负向变化减半，正向变化也减半）
elif 当前关系 > +60（友好/同盟）:
    关系修正 = ×1.2（正向变化加速，负向变化也加速）

情境修正：
- 目标处于内乱：正向变化 +50%
- 目标面临外部威胁：正向变化 +30%
- 瓦希德近期军事胜利：正向变化 +20%
- 瓦希德近期军事失败：正向变化 -30%
```

#### 公式2：外交行动成本
```
行动成本 = 基础成本 × 关系修正 × 外交难度修正

关系修正：
- 敌对：×1.5
- 冷淡：×1.2
- 一般：×1.0
- 友好：×0.8
- 同盟：×0.6

外交难度修正（目标相关）：
- 黄金领：×1.5（最难）
- 圣火序：×1.3
- 金帐合众：×1.1
- 北境共主：×1.0
- 东土商盟：×0.9
- 中立国：×1.0
```

#### 公式3：联盟防御触发
```
联盟防御触发条件：
- 瓦希德遭受军事攻击（节点被占领、遭受全面战争）
- 攻击方与某友好/同盟势力关系 > +60
- 该势力决定是否响应防御条款

响应效果：
- 同盟势力在下一回合提供：2~5战械 或 等值金叶援助
- 若多个同盟响应：效果叠加
- 响应强度 = f(关系等级, 该势力自身实力)

不触发条件：
- 瓦希德主动攻击他人（先发制人情况除外）
- 攻击方与同盟势力关系 > +80（同盟国不干预）
```

#### 公式4：关系自然波动
```
每5回合自然变化：

if 有持续贸易往来（每回合商盟券>0）:
    关系 += +1（缓慢改善）
elif 长期无外交接触（>5回合无外交行动）:
    关系 -= -1（缓慢恶化）
elif 封锁压力持续（单边封锁以上>3回合）:
    关系 -= -2（负面影响累积）
```

## Edge Cases 边界情况

| 边界情况 | 处理方式 |
|---------|---------|
| 关系变化导致越过正负阈值 | 立即更新状态，触发对应效果（如从友好变冷淡立即失去折扣） |
| 与所有势力关系同时恶化 | 触发"外交孤立"警告事件；封锁压力剧增 |
| 断交后试图恢复关系 | 需要"破冰行动"（特殊外交行动，代价×2） |
| 目标国政权更替（AI内部政治） | 关系在-20~+20范围内随机重置 |
| 同盟国被第三方攻击 | 瓦希德可选择是否响应（触发联盟防御条款评估） |

## Data Interfaces 数据接口

### C1向其他系统提供的接口

```typescript
interface DiplomaticRelationsSystem {
    // 获取与某势力的关系
    getRelation(factionId: FactionId): Relation
    
    // 获取所有势力的关系状态
    getAllRelations(): RelationMap
    
    // 获取外交行动成本
    getActionCost(action: DiplomaticAction, target: FactionId): ResourceCost
    
    // 评估联盟防御触发可能性
    assessAllianceDefense(attacker: FactionId): AllianceDefenseAssessment
    
    // 计算关系对某系统的影响系数
    getRelationModifier(target: FactionId, dimension: 'trade' | 'military' | 'blockade'): number
}

interface Relation {
    factionId: FactionId
    level: RelationLevel
    value: number              // -100 ~ +100
    trend: 'improving' | 'stable' | 'declining'
    lastChangeTurn: number
    specialStatus: 'war' | 'alliance' | 'none'
}
```
